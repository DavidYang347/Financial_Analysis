> **关键判断**：以技术壁垒和转换成本为核心，护城河真实且正处于加宽过程中。AI算力需求和国产替代政策提供了外部驱动力。主要风险是技术迭代速度过快或海外竞争对手加大中国市场投入。

### 2.2 行业格局

| 维度 | 内容 |
| --- | --- |
| **行业生命周期** | 成长期（受益AI算力爆发和国产替代双重驱动，整体快速扩张） |
| **竞争格局** | 高端市场高度集中，海外巨头主导：2024年海外企业占中国光通信测试仪器市场约84%份额，国产企业仅16%；联讯以9.9%位列全国第三、本土第一[[来源：招股书/新浪财经]](http://money.finance.sina.com.cn/corp/view/vCB_AllBulletinDetail.php?id=11894261&stockid=688808) |

| **替代风险** | 技术路线风险存在但可控：如800G→1.6T→3.2T的迭代中，联讯已提前布局；存储芯片测试领域尚有待突破。政策层面有大力支持 |
| **供需状态** | 供不应求（AI算力驱动数据中心大规模建设，光模块测试仪器需求激增，国产替代进程加速） |
| **价格趋势** | 高端测试仪器价格相对稳定，联讯以性价比优势切入，不走低价路线 |

> **行业结论**：行业处于明显顺风，AI驱动的光模块迭代（400G→800G→1.6T→3.2T）和国产替代政策双重催化，利润池持续扩大。测试环节是"卡脖子"关键环节，景气周期可望持续3-5年以上。[[来源：国金证券研报/网易]](https://www.163.com/dy/article/KRHLBMK605568W0A.html)

## 3. 业绩驱动拆解

### 3.1 增长从哪里来

| 驱动项 | 现状 | 判断 |
| --- | --- | --- |
| **量增** | AI数据中心建设带动光模块测试需求爆发；800G→1.6T渗透率提升；碳化硅功率器件测试设备国内市场快速扩容 | 强 |
| **价增** | 随技术代际（1.6T升级至3.2T）单台设备价值量增加；产品结构向高端迁移，综合单价上行 | 中 |
| **份额提升** | 2024年国产替代加速，海外巨头因监管限制进入难度加大，国产替代系统性利好；联讯份额持续提升 | 强 |
| **新品/新业务** | 存储芯片测试（DRAM/HBM）、车规芯片测试已获Demo订单，3.2T光模块测试预研启动，构成第二增长曲线[[来源：东方财富研报]](https://pdf.dfcfw.com/pdf/H3_AP202604091821101422_1.pdf) | 强 |
| **并购贡献** | 目前无明显并购动作，依靠内生增长 | 弱 |

> **核心增长逻辑**：未来1-3年核心增长来源 = AI算力驱动的光模块测试量增（800G→1.6T渗透+3.2T预研）× 国产替代份额提升 × 存储/车规芯片测试新赛道突破。中性预期下2026年净利有望翻2-3倍。

### 8.2 主要矛盾
