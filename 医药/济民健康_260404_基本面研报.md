## 一、商业模式与竞争优势

### 1.1 业务概述

| 字段 | 内容 |
|------|------|
| **主营业务** | 医疗器械（安全注射穿刺器械、预充式导管冲洗器等）研发生产销售；医疗健康服务（博鳌国际医院、鄂州二医院）；大输液产品（3000ml氯化钠注射液、平衡盐溶液等）研发生产销售。旗下品牌"双鸽"在国内注射器领域有30余年历史。 [[来源：百度百科]](https://baike.baidu.com/item/%E6%B5%99%E6%B1%9F%E6%B5%8E%E6%B0%91%E5%88%B6%E8%8D%AF%E6%9C%89%E9%99%90%E5%85%AC%E5%8F%B8/1778687) |
| **收入来源构成** | 医疗服务板块41.79%（32,136万元）、大输液板块35.69%（27,682万元）、医疗器械板块22.02%（17,081万元）[[来源：2024年报]](https://static.cninfo.com.cn/finalpage/2025-04-29/1223368738.PDF) |
| **地理收入拆分** | 以国内为主；医疗器械有较大比例出口（主要出口美国，通过美国RTI公司代工协议），受美国关税冲击外销下滑约40%。西班牙子公司LINEAR CHEMICALS生产化学原料药。 [[来源：2025年半年报]](http://qxb-pdf-osscache.qixin.com/AnBaseinfo/88d183868ef89c7282859cba8fe174a2.pdf) |
| **核心客户群体** | 医疗器械板块：终端医院（B端）及经销商；医疗服务板块：个人患者（C端）；大输液板块：医院（B端）。前五大客户收入合计13,908万元，占年度总额17.93%，客户分散度高。 [[来源：2024年报]](https://static.cninfo.com.cn/finalpage/2025-04-29/1223368738.PDF) |
| **收入模式** | 医疗器械和大输液：以销售为主（直销+经销混合）；医疗服务：项目制及按次收费（医院诊疗服务费）。 |
| **价值链位置** | 医疗器械处于产业链中游制造端，话语权中等偏弱；医疗服务处于终端服务提供者，局部有定价权（博鳌国际医院特色高端定位）；大输液处于产业链中游。 |
| **季节性特征** | 医疗服务具有一定季节性（秋冬就诊高峰）；大输液和医疗器械季节性不明显。博鳌国际医院因地处海南，受台风等自然灾害影响（2024年Q4受台风影响较大）。 [[来源：2024年报]](https://static.cninfo.com.cn/finalpage/2025-04-29/1223368738.PDF) |
| **产品/服务生命周期** | 注射穿刺器械：成熟期；大输液：成熟期；医疗服务（特别是博鳌国际医院再生医学/创新药业务）：导入期。整体处于多元化转型阶段，核心传统业务成熟但增长乏力。 |

**波特五力量化评估**：

| 竞争力量 | 强度 | 关键判断依据 |
|---------|------|------------|
| 现有竞争者的竞争强度 | 高 | 医疗器械和大输液均为充分竞争市场，国家集采持续压价，毛利率下行压力持续存在 |
