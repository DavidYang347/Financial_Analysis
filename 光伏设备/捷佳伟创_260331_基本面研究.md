## 一、商业模式与竞争优势

### 1.1 业务概述

| 字段 | 内容 |
|------|------|
| **主营业务** | 太阳能电池生产工艺设备的研发、制造和销售，全面覆盖PERC、TOPCon、HJT、XBC、钙钛矿及钙钛矿叠层等主流高效电池技术路线，同时向半导体设备和锂电真空专用设备领域延伸。[[来源：2024年年度报告摘要]](https://file.finance.qq.com/finance/hs/pdf/2025/04/24/1223243137.PDF) |
| **收入来源构成** | 工艺设备（86.16%）、自动化配套设备（10.66%）、配件及其他（3.18%）。2024年工艺设备收入162.74亿元（+130.44%），自动化设备20.13亿元（+76.99%），配件6.00亿元（+12.40%）。[[来源：乐居财经研报]](https://m.lejucaijing.com/report-165942.html) |
| **地理收入拆分** | 主要以国内为主，海外业务持续拓展。截至2025年Q3，海外订单占在手订单比例约15%，较2024年增长6个百分点；马来西亚工厂已建设，积极开拓欧亚市场。[[来源：东方财富估值报告]](https://caifuhao.eastmoney.com/news/20251030235034516572630) |
| **核心客户群体** | B端为主，核心客户为国内外大型光伏电池生产企业，包括通威股份、晶澳太阳能、隆基绿能、晶科能源等头部光伏企业，以及海外光伏客户。[[来源：OFweek年报解读]](https://solar.ofweek.com/2025-04/ART-260003-8500-30661900.html) |

| **价值链位置** | 处于光伏产业链中游（设备制造），向下游电池片生产企业供应核心工艺设备。话语权较强，具备整线交付能力，客户更换成本高。 |

> **护城河宽度评级**：一般偏宽护城河
>
> **分析说明**：捷佳伟创的核心护城河来源于全技术路线布局能力（全球唯一覆盖TOPCon/HJT/XBC/钙钛矿全技术路线整线供应商）和多个关键设备品类的高市占率（制绒/清洗设备70-80%，TOPCon核心设备>50%）。护城河的耐久性有一定挑战：光伏行业技术迭代速度快，若新技术路线（如BC电池）大规模替代TOPCon，且公司在BC设备领域布局不足，将构成护城河侵蚀风险。此外，竞争对手迈为股份在HJT领域具有更强优势，随HJT渗透率提升，竞争格局可能发生结构性变化。[[来源：OFweek竞争分析]](https://mp.ofweek.com/solar/a556714164577)
