## 一、商业模式与竞争优势

### 1.1 业务概述

| 字段            | 内容                                 |
| ------------- | ---------------------------------- |
| **主营业务**      | 锂离子电池正极材料与钠离子电池正极材料的研发、生产和销售，应用于新能源汽车动力电池、消费电子及储能，并积极布局固态电池关键材料（固态电解质、富锂锰基等）[[来源：证券之星]](https://stock.quote.stockstar.com/corp/brief_688707.shtml)                       |
| **收入来源构成**    | 以三元正极（NCM，中镍/中高镍/高镍/超高镍）为主要收入来源，兼有钴酸锂、复合三元；高镍及超高镍销量占比已达50%；钠电正极材料已装车出货但占比尚小[[来源：证券之星F10]](http://stock.stockstar.com/AN2025090200030735.shtml)        |
| **地理收入拆分**    | 以国内新能源汽车及消费电子客户为主；海外客户端（丰田、三星等）处于百公斤级验证阶段，尚未形成规模收入 [[来源：巨潮咨询2025中报摘要]](https://static.cninfo.com.cn/finalpage/2025-08-16/1224499222.PDF) |
| **核心客户群体**    | B 端动力电池厂商（宁德时代、比亚迪、孚能科技、多氟多、天津力神、珠海冠宇、微宏动力等）[[来源：深圳热线]](http://licai.szonline.cn/cj/2022/1019/41942.html)               |
| **收入模式**      | 项目/订单制的大宗工业品销售（按合同/框架协议按月或按批次发货，非订阅制、非抽佣）[[来源：证券之星]](https://stock.quote.stockstar.com/corp/brief_688707.shtml)         |

| **产品/服务生命周期** | 传统三元正极在国内已处于成熟期后段（份额被磷酸铁锂挤压），高镍/超高镍处于成熟期、钠电正极及固态电解质处于导入期，公司整体面临明显的产品结构更新换代压力[[来源：时代周报]](https://www.time-weekly.com/wap-article/326177)       |

> **护城河宽度评级**：窄护城河（偏弱）
>
> **分析说明**：公司核心竞争力集中在高镍/超高镍单晶工艺、大客户长周期认证关系以及央企背景带来的产业资源，但成本优势和规模效应尚未兑现，三元主赛道已被磷酸铁锂侵蚀，护城河正在被"技术路线变迁"这一更强的外部力量削弱。能否凭借钠电和固态电池关键材料重建护城河，是未来3年核心看点。

- 无频繁非主业资产处置
- 表外负债公开披露较为清晰，风险可控

### 9.2 经营风险

- **客户集中度过高**（宁德时代估约40-60%），失去大客户份额风险严重
- 单一产品/业务（三元正极）依赖，转型钠电与固态尚需时间
- 团队在销售与多元化方面较弱
- 供应链：锂、镍、钴原料价格波动大，供应商集中
- **产能利用率持续低位**（估<40%），固定成本摊薄压力显著
- 技术路线迭代风险：若磷酸锰铁锂体系成为A00级主流，钠电需求可能低于预期
- 海外业务依赖单一客户或区域的风险尚未突出（目前占比小）

### 9.3 行业与外部风险

- 三元锂电池路线长期份额继续收缩的可能性不低
- 行业价格战持续，毛利率修复的时点不确定
- 固态电池若真正商业化，将对现有三元正极形成"再替代"冲击，但公司提前布局属对冲
- 宏观经济下行，新能源车销量不及预期传导至上游
- 原材料价格波动超预期
- 美国IRA法案、欧洲电池法案对中国正极材料的出口限制

### 9.4 治理与合规风险

- 未见重大诉讼/仲裁
- 央企背景+稳定治理，实控人个人风险低
- 董事会换届刚完成，新任董事长左才凤战略表述尚待观察
- 环保合规风险（化工类制造业天然存在）
- 财务信息可信度较高，但需关注"连续亏损年末是否会出现大额计提"

### 12.2 投资决策清单

**商业模式与护城河**

- 一句话解释：公司靠"为锂电池厂商（尤其宁德时代）提供三元/钠电/固态等高端正极材料"赚钱
- 护城河在未来5年能否维持或加深：取决于钠电与固态能否接棒三元，**不确定性较高**
- 有没有某个竞争对手/技术变革正在侵蚀其核心优势：**有——磷酸铁锂、磷酸锰铁锂、头部容百/当升的多元化战略**

**财务质量**

- 净利润是否有真实现金流支撑：**2023年有，2024-2025年已无（均为负）**
- ROE是否稳定在15%以上：**否（当前为负）**
- 过去3年是否出现过财务重述、审计意见非标或频繁更换审计师：**没有**

**估值合理性**

- 当前隐含预期是否合理：**处于中性偏乐观假设区间**
- 溢价/折价是否被基本面差异解释：**部分可解释（钠电/固态期权），部分高估（当前盈利能力）**

**管理层与治理**

- 业绩承诺兑现率：**低（2022年市场一致预期严重高估了2023-2025年业绩）**
- 管理层增减持行为：**较少/不显著**
- 大股东质押比例：**无显著风险**

**风险与安全边际**

- 最悲观情景下行空间：**至10元左右（对应PB 1.25x），跌幅约-30%**
