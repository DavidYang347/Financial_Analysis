## 一、商业模式与竞争优势

### 1.1 业务概述

| 字段 | 内容 |
|------|------|
| **主营业务** | 各类泵的研发、生产、销售及提供相关服务。产品覆盖民用农泵、家用屏蔽泵（热水循环泵）、工业泵（化工屏蔽泵）、液冷泵（新能源/算力）、商用泵五大板块，有200余种规格型号。企业精神"全心全意做好泵——给系统一颗强劲的心" [[来源：大元泵业2025半年报]](https://file.finance.qq.com/finance/hs/pdf/2025/08/26/1224563435.PDF) |
| **收入来源构成** | ①民用泵（农泵）：约占总收入30～35%，为农灌、居民生活等提供传统泵类；②家用屏蔽泵：约占35～40%，主要为壁挂炉/热泵暖通配套，是历史最高毛利率业务；③工业泵：约占7～10%，主要为化工领域屏蔽泵；④液冷泵：2025H1约占7.2%（约0.7亿），高速增长；⑤雷客商泵：新兴业务，2025H1同比+80%以上 [[来源：同花顺经营分析]](http://basic.10jqka.com.cn/603757/operate.html) |
| **地理收入拆分** | 国内外双渠道，外销占较大比重（欧洲热泵市场为主要境外市场）。2024-2025年受欧洲热泵去库存冲击，外销收入明显承压；工业泵外贸及液冷新兴业务弥补部分缺口。国内市场农泵受政策、消费景气影响，2024年内贸整体偏弱 [[来源：大元泵业2024年报摘要]](https://file.finance.qq.com/finance/hs/pdf/2025/04/28/1223347180.PDF) |
| **核心客户群体** | B2B为主。民用泵通过经销商体系覆盖全国；家用屏蔽泵直供壁挂炉/热泵主机厂（欧洲、国内品牌）；工业泵面向化工企业；液冷泵客户包括维谛技术、英维克、中兴通讯、曙光数创、同飞股份、零跑汽车、中车株洲、比特大陆等 [[来源：证券之星互动平台]](https://stock.quote.stockstar.com/603757.shtml) |
| **收入模式** | 一次性产品销售为主（设备+配件），少量维修售后服务。工业泵领域有配件/售后维修收入；液冷领域项目制/框架协议逐步增加 |
| **价值链位置** | 产业链中游（泵制造商），原材料采购上游（铸铁件、漆包线、硅钢片等），产品供应至下游系统集成商或直接主机厂。屏蔽泵细分领域国内领先，话语权中等偏强 |
| **季节性特征** | 农泵具有明显季节性，Q2（春耕备播）为传统旺季，Q1为淡季；家用屏蔽泵跟随欧洲暖通采购节奏，Q1-Q2通常备货，Q3-Q4出货；整体而言H1略重于H2 |
| **产品/服务生命周期** | 传统民用农泵处于成熟期，市场竞争激烈；家用屏蔽泵为成长期（节能泵渗透率提升）；工业屏蔽泵成熟期；液冷泵处于导入-成长期，是未来增长最核心看点 |

> **护城河宽度评级**：一般（偏窄护城河）
>
> **分析说明**：大元泵业在屏蔽泵细分领域具备一定技术积累和客户基础，但整体护城河并不宽。民用农泵同质化严重，工业泵高端市场仍被外资主导（威乐、格兰富等），液冷泵属于新赛道尚在爬坡。护城河的核心支撑在于屏蔽泵制造工艺积累与客户关系，侵蚀风险主要来自竞争对手在液冷领域的追赶。

### 1.3 行业格局（波特五力分析）

| 字段 | 内容 |
|------|------|
| **行业生命周期** | 传统泵行业整体处于成熟期；液冷泵细分处于成长期 |
| **市场集中度** | 国内泵行业高度分散，CR5估计不足20%，单一厂商市占率均较低。外资品牌（威乐、格兰富）主导高端细分市场 [[来源：可转债上市公告书]](http://static.cninfo.com.cn/finalpage/2022-12-30/1215484209.PDF) |
| **主要竞争对手** | ①利欧股份（002131）：民用水泵+工业泵+数字营销，营收规模远大于大元；②凌霄泵业（603286）：民用水泵，2021年营收20.63亿；③君禾股份（603139）：民用水泵（潜水泵、花园泵等），规模偏小；④南方泵业（002271）：工业泵；⑤外资：威乐（德国）、格兰富（丹麦）主导高端工业/家用领域 [[来源：可转债上市公告书]](http://static.cninfo.com.cn/finalpage/2022-12-30/1215484209.PDF) |
| **潜在替代威胁** | 液冷泵有被其他技术路线（风冷）替代的可能，但行业趋势是液冷渗透率持续提升（GB200/300高密机柜推动）；传统农泵面临更节能产品迭代压力 |
| **公司市场份额** | 整泵产量约占全国2.5～3%（2020年数据），屏蔽泵细分国内领先。近年趋势：民用板块份额相对稳定，液冷板块从0增长 [[来源：可转债上市公告]](http://static.cninfo.com.cn/finalpage/2022-12-30/1215484209.PDF) |

**波特五力量化评估**：

| 竞争力量 | 强度 | 关键判断依据 |
|---------|------|------------|
| 现有竞争者的竞争强度 | 高 | 国内泵企数量众多，民用泵价格战频繁；高端工业泵外资强势；液冷泵众多新进入者 |
| 新进入者威胁 | 中 | 民用泵门槛低，新进入者多；工业泵/液冷泵需要认证积累，门槛中等 |
| 替代品威胁 | 低-中 | 传统泵短期无颠覆性替代；液冷泵路线内竞争加剧（屏蔽式vs浸没式vs喷淋式） |
| 供应商议价能力 | 中 | 主要原材料为铸铁件、漆包线、硅钢片等大宗，价格受市场影响，公司采购规模一般，话语权中等 [[来源：同花顺经营分析]](http://basic.10jqka.com.cn/603757/operate.html) |
| 买方议价能力 | 中-高 | 大客户（热泵主机厂、液冷集成商）议价能力较强；经销商分散化，单一经销商议价能力低 |

> **行业吸引力综合评估**：传统民用/农泵行业吸引力偏弱（分散、竞争激烈、毛利承压）；家用屏蔽泵受欧洲市场波动影响较大；工业泵受化工行业资本开支周期影响显著；液冷泵赛道吸引力高但竞争正在快速激化。整体而言，公司处于行业中游位置，短期盈利能力承压，中长期向液冷/高端转型是核心逻辑。

### 11.4 当前核心看点总结

**看点一：液冷泵从"小而美"到"规模化放量"的验证**
> 当前液冷泵收入约1亿+/年（2025H1 0.7亿），客户名单持续扩大（维谛、英维克、曙光数创、比特大陆等），但体量仍小。若2026年能突破3-5亿收入规模，结合较高毛利率，将实质性改变公司整体盈利结构。这是当前最值得跟踪的核心看点，也是当前估值溢价的主要依据。

**看点二：家用屏蔽泵复苏节奏——欧洲热泵市场超预期是关键**
> 欧洲热泵市场经历2023-2024年深度调整，2025H1已有明显复苏迹象（节能泵销量回升）。若欧洲暖通市场2026年进入正常补库通道，家用屏蔽泵这一历史上毛利率最高业务将重新贡献高质量利润，是拉动整体利润率回升的最重要传统业务变量。

**看点三：大元转债赎回后的财务结构改善**
> 4.5亿可转债赎回后，年化利息支出约2000-2700万将消失，相当于直接增厚净利润约15-20%（以2025年低基期测算）。这是一个确定性较强的利润改善催化剂，预计2026年内实现。
