## 一、商业模式与竞争优势

### 1.1 业务概述

| 字段 | 内容 |
|------|------|
| **主营业务** | 精密温控节能解决方案与产品提供商。核心产品涵盖**机房温控节能产品**（数据中心精密空调、液冷解决方案）、**机柜温控节能产品**（储能温控、通信户外柜空调）、**客车空调**（新能源客车及公路客车空调）、**轨道交通列车空调及服务**（上海、苏州等地铁项目）。 [[来源：雪球]](https://xueqiu.com/S/SZ002837) |
| **收入来源构成** | **机房温控节能**：24.41亿元，占53.19%，同比+48.83%；**机柜温控节能**：17.15亿元，占37.38%（其中储能温控约15亿元，+22%），同比+17.03%；**其他业务**（客车空调、轨交等）：约4.33亿元，占约9.43%。（2024年报数据）[[来源：新浪财经]](https://finance.sina.com.cn/stock/aigc/stockfs/2025-04-22/doc-inetzeny4278006.shtml) |

| **核心客户群体** | B端为主：大型互联网公司（腾讯、阿里、百度等）、通信运营商（中国移动、电信、联通）、IDC运营商（秦淮数据、万国数据、数据港）、算力设备厂商（华为、中兴）、储能系统客户（比亚迪、阳光能源）等。[[来源：未来智库]](https://www.vzkoo.com/read/2024062499cdf01828a55c39bc3c1d84.html) |

### 1.2 核心护城河评估

| 护城河类型 | 是否具备 | 评估说明 |
|-----------|---------|---------|
| 品牌壁垒 | ✅ | 国内数据中心温控"双龙头"之一，华为/中兴/腾讯/阿里等知名客户验证，行业内高知名度 |
| 成本优势 | ✅（一般） | 规模采购铜铝等原材料有一定优势，但受大宗商品价格波动影响；毛利率28.75%处于行业中等偏上水平 |
| 网络效应 | ❌ | 温控设备行业不具有典型网络效应 |
| 转换成本 | ✅ | 数据中心液冷系统的运维、专有工质（SoluKing）、系统集成深度，客户更换成本较高；运营商集采框架协议构成粘性 |
| 专利/资质壁垒 | ✅ | 拥有专利1159项（截至2024H1）、软件著作权103项，参与制定多项国家/行业标准；液冷全链条已通过英特尔验证，UQD产品列入英伟达MGX生态合作伙伴。[[来源：东方财富研报]](https://pdf.dfcfw.com/pdf/H3_AP202410041640171635_1.pdf) |
| 规模经济 | ✅ | 产能扩张带来采购规模优势，研发平台可共享至多业务线；但当前规模效应的体现在毛利率维度尚不突出 |
| 数据壁垒 | ❌ | 暂未积累显著数据资产 |
| 生态系统绑定 | ✅（初步） | 英维克Coolinside全链条液冷深度绑定英伟达/英特尔生态；长效液冷工质SoluKing与系统绑定，构建轻度生态粘性 |
| 地理/渠道垄断 | ❌ | 无特定区域垄断，但深圳、郑州、马来西亚等多地布局形成全球供应链优势 |

> **护城河宽度评级**：**一般偏宽**
>
> **分析说明**：英维克的护城河主要来自技术积累（全链条液冷、20年行业深耕）、品牌背书（进入英伟达/英特尔生态）和客户转换成本。风冷产品时代护城河较窄（技术成熟，竞争者多），液冷时代英维克凭借先发优势和全链条能力拓宽了护城河，但来自高澜股份、曙光数创、申菱环境及海外巨头Vertiv/维谛的竞争在持续加剧，护城河面临一定侵蚀风险。

| 字段 | 内容 |

| **公司市场份额** | 储能温控约16%（2023），数据中心温控国内约前三名水平（未有精确数据），近3年趋势**上升**。[[来源：智研咨询]](https://www.chyxx.com/industry/1208869.html) |

**波特五力量化评估**：

| 竞争力量 | 强度 | 关键判断依据 |
|---------|------|------------|
| 现有竞争者的竞争强度 | **高** | 液冷赛道入局者持续增多（国内50+家上市储能温控企业），部分项目价格竞争激烈，毛利率整体承压（2024年下降约3.6pct） |
| 新进入者威胁 | **中** | 资本门槛较高，全链条液冷技术积累需时间；但冷板式液冷技术门槛相对较低，仍有新进入者 |
| 替代品威胁 | **中** | 浸没式液冷相对冷板式有一定替代威胁；AI芯片节能设计改善或降低未来散热需求 |
| 供应商议价能力 | **中** | 压缩机、铜铝原材料价格波动2024年影响毛利率；但英维克规模采购有一定优势 |
| 买方议价能力 | **中高** | 互联网大厂/运营商采购体量大，集采模式具备较强议价能力；大客户占比较高存在压价风险 |

> **行业吸引力综合评估**：数据中心液冷温控行业在AI算力浪潮驱动下呈现高成长性，政策（双碳+低PUE要求）加持需求端，未来3-5年CAGR预计在30%以上。英维克凭借先发布局和全链条能力占据有利竞争地位，但随竞争加剧，行业整体盈利性将持续承压，个体竞争优势的护城河持续性是核心考量。

### 11.4 当前核心看点总结

**看点一：AI液冷渗透率加速，英维克全链条壁垒最强**
> 英伟达GB200/GB300单机柜功耗突破140kW，液冷从可选项升为必选项，渗透率有望从当前10-15%快速提升至2026年50%+。英维克作为国内全链条液冷开创者（冷板+快接头+Manifold+CDU+工质+冷源全覆盖），且已进入英伟达MGX生态系统，是国内最具竞争力的全链条液冷供应商，有望占据市场份额提升的最大份额。2025年全年液冷相关营收预计将实现100%+增长，是最重要的业绩弹性来源。

**看点二：2025年全年报验证业绩拐点**
> 2025年H1和Q3均受项目验收节奏慢、大宗商品涨价影响，利润增速明显低于营收增速。Q4为全年最旺季，历史上Q4收入占全年30-35%，且验收集中导致净利润往往集中释放。若2025年全年净利润确认≥6亿元（约+30%以上），将大幅修复市场对"增收不增利"的担忧，形成估值修复的关键驱动。2025年全年报（2026年4月底发布）是当前最重要的催化节点。

**看点三：海外产能落地开启第二成长曲线**
> 泰国工厂计划2026年Q3投产（年产能5万台液冷机柜，专供东南亚英伟达/Meta数据中心），马来西亚工厂2025年底产能占比升至25%。关税壁垒下，本地化制造能力是打入海外市场的关键筹码。若2026年海外业务收入占比从当前14%提升至20%+，将开启新的估值溢价空间。海外工厂投产进度是关键催化，而关税政策走向是最大不确定因素。
