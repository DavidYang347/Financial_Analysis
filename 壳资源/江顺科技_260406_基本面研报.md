## 一、商业模式与竞争优势

### 1.1 业务概述

| 字段 | 内容 |
|------|------|
| **主营业务** | 铝型材挤压模具及配件（核心）、铝型材挤压配套设备、精密机械零部件三大业务线的研发、设计、生产和销售 [[来源：上市首日公告]](https://www.giansun.com/attached/file/20250425/20250425104241_14408.pdf) |

| 护城河类型 | 是否具备 | 评估说明 |
|-----------|---------|---------|
| 品牌壁垒 | ✅ | 深耕铝型材挤压模具20余年，已与豪美新材、鑫铂股份等头部铝型材企业形成长期合作，品牌认知度在细分领域较强 |
| 成本优势 | ✅ | 规模化生产带来一定采购议价优势，且产品高度定制化，竞争维度不单是价格 |

| 转换成本 | ✅ | 铝型材挤压模具定制化程度高，客户与公司在设计图纸、参数调试上深度绑定，迁移成本较高；设备业务亦具有一定售后服务黏性 |
| 专利/资质壁垒 | ✅ | 具备自主设计创新体系、精密制造体系，拥有一批专利，研发费用率持续在4.5%-5.1%之间 [[来源：上市首日公告]](https://www.giansun.com/attached/file/20250425/20250425104241_14408.pdf) |
| 规模经济 | ✅ | 目前属于细分领域领先企业，规模扩张带来一定效率提升，但行业以中小企业为主，集中度低 |
| 数据壁垒 | ❌ | 无数据资产积累 |
| 生态系统绑定 | ❌ | 无生态系统 |
| 地理/渠道垄断 | ❌ | 暂无排他性区域布局 |

> **护城河宽度评级**：一般
>
> **分析说明**：公司在铝型材挤压模具细分领域积累了技术和客户壁垒，转换成本和品牌认知是主要护城河，但行业集中度整体较低（"以中小企业为主"），护城河宽度有限。核心风险在于：技术壁垒并非不可跨越，竞争对手跟进能力较强，若大客户降低集中采购或引入竞对，则话语权将受压。

**波特五力量化评估**：

| 竞争力量 | 强度 | 关键判断依据 |
|---------|------|------------|
| 现有竞争者的竞争强度 | 中 | 行业分散，中小企业众多；但公司定制化高端产品受价格战影响相对有限；H1 2025 毛利率34.43%仍维持合理水平 |
| 新进入者威胁 | 中 | 精密制造存在一定技术门槛，需要多年经验积累，但资本门槛不高；高端市场的进入壁垒更高 |
| 替代品威胁 | 低 | 铝型材挤压为成熟工艺，无近期技术替代路线，且"以铝代钢"趋势强化 |
| 供应商议价能力 | 中 | 原材料（钢材类、电气类、五金类）占营业成本约65%，供应商较分散；原材料价格波动有一定滞后性传导 [[来源：上市首日公告]](https://www.giansun.com/attached/file/20250425/20250425104241_14408.pdf) |

> **行业吸引力综合评估**：铝型材挤压模具及设备行业处于稳步成长期，受多元下游拉动，行业空间持续扩大。公司在高端客户群中建立了差异化优势，但行业集中度低使整体盈利能力提升有限。作为行业领先者，公司盈利能力（模具业务毛利率40%+）显著优于行业平均水平，体现一定竞争优势，但可持续性仍需跟踪。

- 2025年年报大概率继续大幅负增长（前三季度利润已-40.15%），若年报确认业绩下滑，情绪将受到重大打击
- 概念炒作逻辑（低空经济/航空航天/液冷）已被公司多次辟谣，泡沫脆弱

- 设备业务在手订单交付进度（需持续跟踪季报）
- 境外收入增长能否持续（汇率风险+贸易政策风险）

### 11.4 当前核心看点总结

**看点一：2025年年报业绩（2026-04-28披露）**
> 这是当前最重要的验证节点。基于前三季度已披露的数据（收入-13.68%，利润-40.15%），市场普遍担忧全年业绩继续大幅下滑。若年报数据好于预期（比如Q4环比改善），可能带来短期情绪改善；但若确认大幅下滑，将对当前97倍PE估值产生严重冲击。

**看点二：境外收入的可持续性**
> 公司境外收入是当前唯一强劲的业务亮点（H1 2025+31.91%）。这一趋势若能持续（尤其是东南亚和欧洲市场开拓），将构成真实基本面支撑；若受贸易摩擦或汇率冲击而减速，公司将失去唯一增长引擎，估值压力将进一步加大。

**看点三：设备业务订单交付修复**
> H1 2025设备业务收入大幅下滑35.71%，主因客户基础设施建设延期。在手订单的最终交付时间节点，将直接决定设备业务未来1-2个季度的收入走势，是重要的跟踪变量。
