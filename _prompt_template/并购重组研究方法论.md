# 并购重组预期标的研究方法论

> 并购重组是A股市场最复杂的投资逻辑之一，既有巨大的赔率，也有极高的信息不对称风险。本文档提供一套从信息搜集到定价评估的完整研究框架。

---

## 一、前置模块：信息搜集与时间线整理

在进入分析框架之前，必须先完成一轮系统性的信息搜集，建立对标的并购重组事件的"事实底层"。跳过这一步直接分析，容易被二手信息和市场情绪带偏。

### 1.1 信息搜集渠道矩阵

| 信息类型 | 首选渠道 | 检索关键词 |
|---|---|---|
| 监管公告（法定披露） | 巨潮资讯（cninfo.com.cn） | 股票代码 + "重大资产重组" / "发行股份" / "停牌" |
| 交易所问询函 | 上交所/深交所官网监管动态 | 股票代码 + "问询函" / "关注函" |
| 证监会审核进度 | 证监会官网行政许可栏目 | 公司简称 + "并购重组" |
| 财经新闻与市场解读 | Wind、彭博、财联社、界面、第一财经 | 公司简称 + "重组" / "并购" / "注资" / "借壳" |
| 大股东工商信息 | 天眼查、企查查 | 控股股东/实控人全称，查旗下关联资产 |
| 研究报告 | 东方财富Choice、万得、dfcfw.com | 公司代码 + "并购" / "外延" |
| 社交舆情（辅助参考） | 雪球、东方财富股吧 | 股票代码/简称，注意过滤噪音 |

### 1.3 时间线分析的核心问题

整理完时间线后，必须回答以下问题，再进入后续分析框架：

1. **重组预期是何时开始被市场定价的？** 股价启动时间与首个公告时间的差值，可以判断是否有信息泄露或内幕交易。
2. **信号演变是加速还是停滞？** 公告间隔越来越短 → 推进顺利；长时间无新公告 → 警惕悬而未决风险。
3. **是否有监管介入痕迹？** 问询函数量多、回复周期长，说明监管审查严格。
4. **市场情绪是否已过热？** 若舆情中充斥着"必成"、"内幕消息"等表述，市场预期往往已过度定价。
5. **是否存在历史前车之鉴？** 该公司/大股东是否有过重组失败记录（查历史公告）？

## 二、判断并购重组的类型与驱动力

在做任何深入分析之前，必须先搞清楚这是哪种性质的重组，因为研究路径完全不同：

| 类型 | 核心驱动 | 典型特征 | 成功率参考 |
|---|---|---|---|
| **控股股东资产注入** | 大股东有优质资产承诺装入上市体 | 大股东持股高、历史有承诺 | 中等 |
| **国资整合** | 地方/央企平台整合同业资产 | 实控人为国资委系 | 较高 |
| **产业并购** | 上市公司主动外延扩张 | 公司账上现金充裕、有战略意图 | 中等 |
| **壳资源借壳** | 标的本身为壳，有新资产借壳进入 | 主业萎缩、市值小、股权分散 | 低且信息极不对称 |
| **要约收购/私有化** | 大股东或战略投资者溢价收购 | 股价长期低迷、PB低 | 视价格 |
| **困境重整** | 破产重整引入战投 | ST/退市边缘、大量债务 | 极低，高度博弈 |

**判断清楚类型，才能确定你的胜率和赔率是否匹配。**

## 三、第一层：重组必要性与动机分析（Why）

> **核心目标：** 判断重组是"真实需求驱动"还是"讲故事套利"，以及各方利益诉求的强烈程度。动机越强、各方利益越一致，推进成功的概率越高。

---

### 3.1 上市公司侧动机分析

**怎么找资料：**
- 巨潮资讯下载最近3年年报，通读"管理层讨论与分析"（MD&A）章节
- 搜索公司历年业绩说明会纪要（投资者关系活动记录）
- 查看同花顺F10 → 股本股东 → 大股东质押比例

**怎么分析：**

| 分析维度 | 信号指标 | 判断逻辑 |
|---|---|---|
| 主业衰退压力 | 营收/净利润连续2年以上下滑 | 主业失速 → 重组救壳动机强 |
| 壳价值保护 | 市值<30亿、资产负债率<40%、历史无大额商誉 | 干净的壳更容易被注入/借壳 |
| 融资需求 | 近2年资本开支大但自由现金流为负 | 需要通过定增配套融资补血 |
| 大股东质押压力 | 质押比例>60%且股价跌近警戒线 | 重组提振股价的紧迫性极高 |
| 管理层表态 | 历次业绩说明会是否主动提及外延并购意图 | 主动表态是真实意图的正面信号 |

**实操技巧：** 在巨潮资讯搜索公司名称 + "同业竞争"、"资产注入"，筛选IPO时的历史承诺文件，这是最硬的动机证据。

---

### 3.2 大股东/注入方侧动机分析

**怎么找资料：**
- 天眼查/企查查：搜索控股股东全称，查其旗下全部关联企业（重点关注未上市资产）
- 巨潮资讯：搜索大股东名称，查其发布的承诺函、历次股东大会决议
- 工商局网站或天眼查：查大股东旗下资产的注册资本、成立时间、经营范围

**怎么分析：**

**第一步：绘制大股东资产图谱**

```
实控人
├── 上市公司（当前研究标的）
├── 未上市子公司A（经营范围、收入规模？）
├── 未上市子公司B（是否与上市公司同业？）
└── 参股公司C（是否有注入潜力？）
```

重点关注：未上市资产中，是否有与上市公司主业高度相关、且规模显著大于上市公司的业务体。这是"同业竞争"承诺下最有可能注入的资产。

**第二步：判断大股东注入的财务动机**

| 动机类型 | 典型信号 | 含义 |
|---|---|---|
| 解决同业竞争承诺 | IPO/再融资招股书中有明确时限承诺 | 硬性催化剂，有明确deadline |
| 大股东缺钱套现 | 大股东股权频繁质押、旗下企业债务到期 | 通过重组拉高股价后减持套现 |
| 资产上市变现 | 未上市资产规模大、盈利强 | 大股东希望通过注入实现未上市资产的流动性 |
| 国资考核压力 | 国资委对下属上市公司资产证券化率有要求 | 政策强制驱动，确定性最高 |

---

### 3.3 政策环境分析

**怎么找资料：**
- 证监会官网：查"并购重组"专题，关注最新监管表态
- 国资委官网：查央企/地方国资整合的专项规划
- 发改委/工信部：行业整合政策文件

**怎么分析：**

当前（2024年后）政策环境已明显松绑，重点关注：
- **"并购六条"**：支持A股上市公司跨行业并购、支持未盈利资产注入、提高重组定价灵活性
- **国资"一企一上市"政策**：推动央企、地方国企整合同类资产至单一上市平台
- **行业整合政策**：军工（军民融合）、半导体（国产替代）、新能源等赛道的整合均有政策背书

**动机综合评分（自建）：**

```
重组动机强烈程度评分（满分10分）：
上市公司侧：
  主业衰退压力（0-2分）：___
  大股东质押压力（0-2分）：___
大股东侧：
  体外资产规模与质量（0-2分）：___
  历史承诺硬约束（0-2分）：___
政策侧：
  行业整合政策支持（0-2分）：___

总分：___/10（7分以上认为动机充足）
```

## 四、第二层：重组可行性分析（Can）

> **核心目标：** 排除"想做但做不了"的情况。很多重组预期最终落空，不是因为动机不强，而是遭遇了财务、监管或股权层面的硬性障碍。

---

### 4.1 财务可行性

**怎么找资料：**
- 巨潮资讯：下载最近3年年报、半年报，重点看资产负债表和附注
- 同花顺F10 / 证券之星：查商誉规模、历次定增记录、负债结构

**核查清单：**

**① 资产负债表是否干净？**

| 项目 | 警戒标准 | 查看位置 |
|---|---|---|
| 商誉 / 总资产 | >20% 需高度警惕 | 资产负债表 + 附注商誉减值测试 |
| 资产负债率 | >70% 制约定增空间 | 资产负债表 |
| 应收账款周转天数 | 较行业均值高出>50% | 财务报表附注 |
| 存货跌价准备 | 是否有大额计提 | 财务报表附注 |

商誉过高意味着：一旦被收购资产业绩不达标，大额减值将严重拖累上市公司净利润，使重组后的整合故事难以为继。

**② 是否有定增空间？**

监管要求：再融资间隔原则上不少于18个月（科创板/创业板有所差异）。
- 查最近一次定增完成时间（巨潮公告搜索"非公开发行"）
- 计算是否已满足间隔要求
- 评估当前股价是否高于历史定增价（否则参与方可能亏损，新定增吸引力低）

**③ 标的资产估值压力测试**

如果注入资产规模已大致可知（来自工商信息或媒体报道），进行简单估算：

```
若注入资产净利润 = X亿元
给予合理PE = Y倍（参考同类资产二级市场估值）
则注入资产估值 ≈ X × Y 亿元

对应新增股份 = 估值 / 当前股价（假设以市价定增）
稀释比例 = 新增股份 / (原股本 + 新增股份)

若稀释比例 > 30%，需关注原股东接受度
```

---

### 4.2 监管可行性

**怎么找资料：**
- 证监会官网"行政许可"栏目：查同类重组历史审核通过率和审核周期
- 交易所监管动态：查是否有针对该公司的存量问询函未回复
- 搜索"XXX 重组 审核 问询"，了解监管关注点

**重点核查项：**

**① 公司资质障碍**

| 障碍类型 | 具体情形 | 影响 |
|---|---|---|
| ST/\*ST 状态 | 公司被特别处理 | 重组审核周期大幅延长，需先摘帽 |
| 立案调查 | 公司或实控人被证监会立案 | 审核期间原则上中止，几乎无法推进 |
| 财务造假记录 | 历史上有会计差错更正或违规记录 | 监管对信披质量高度关注 |
| 科创板特殊规则 | 科创板重组要求标的资产科创属性 | 跨行业注入受到更严格限制 |

**② 交易结构合规性**

- 是否构成借壳（控制权变更 + 注入资产规模超过上市公司）？借壳按IPO标准审核，周期更长
- 是否涉及关联交易？关联交易须经独立股东表决且信披更严格
- 是否需要反垄断申报（市场份额门槛）？

**③ 实操判断方法**

在证监会官网"并购重组委审核结果"栏目中，搜索与目标重组类型相近的历史案例（同行业、同规模、同交易结构），统计通过率，识别监管的核心关注点（通常集中在估值公允性、业绩承诺合理性、信息披露完整性三方面）。

---

### 4.3 股权结构可行性

**怎么找资料：**
- 同花顺F10 → 股本股东 → 查十大股东持股比例
- 巨潮公告 → 搜索"股权转让"、"协议转让"，了解近期股权变动

**分析要点：**

**① 大股东控制力是否足够？**

重组方案须经股东大会特别决议通过（通常需出席股东>2/3赞成）。若大股东持股比例<30%，需要联合其他股东，阻力显著增加。

**② 是否存在"野蛮人"风险？**

若前十大股东中有持股>5%的机构投资者（尤其是与大股东利益不一致的财务投资人），需评估其表决态度。有时大型机构会对估值不公允的重组方案投反对票。

**③ 股权结构稳定性**

- 是否有大量股东处于限售期即将解禁？解禁后减持压力大，不利于重组推进期间的股价维稳
- 最近12个月内是否有大股东减持行为？减持与重组往往信号相悖

## 五、第三层：重组节奏与催化剂追踪（When）

> **核心目标：** 识别当前处于哪个推进阶段，判断催化剂是否真实临近，避免在"预期打满但无落地信号"时入场，承担无谓的时间成本。

---

### 5.1 信号层级识别

**怎么找资料：**
- 巨潮资讯：公司公告全文检索（关键词："停牌"、"重大事项"、"筹划"）
- 上交所/深交所：监管动态 → 问询函及回复
- 财联社、界面新闻：实时快讯

**信号层级完整图谱（从弱到强）：**

```
【Level 1 - 弱信号：市场猜测阶段】
├── 股价无故异动，成交量较均值放大2倍以上
├── 大股东/董监高集中停止减持（查减持公告）
├── 媒体出现"据悉"、"消息人士称"类报道
└── 股吧/雪球出现大量重组讨论但无实质公告

【Level 2 - 中信号：公司有所动作】
├── 公告"停牌，筹划重大资产重组"
├── 公告"聘请XX为独立财务顾问"（FA聘用）
├── 公告"与XX签署战略合作框架协议"
├── 控股股东/实控人变更（协议受让股权）
└── 公司回复交易所"不知悉影响股价的重大信息"（排查性问询）

【Level 3 - 强信号：实质性推进】
├── 证监会正式受理并购重组申请
├── 并购重组委安排审核会议（上会时间披露）
├── 交易所问询函回复中披露方案核心条款
└── 公司披露重组预案/草案

【Level 4 - 落地信号：结果确定】
├── 并购重组委审核通过/否决
├── 股东大会表决通过
└── 完成工商变更登记，资产交割完成
```

**信号判断方法：** 将当前标的对照上述层级，判断处于哪个阶段。Level 1建仓的赔率最高但胜率最低；Level 3介入胜率较高但上涨空间已被压缩；Level 2往往是综合性价比最佳的介入窗口。

---

### 5.2 关键时间节点分析

**外部窗口期（监管层面）：**

| 时间节点 | 窗口类型 | 原因 |
|---|---|---|
| 3月两会后（4-6月） | 政策落地窗口 | 监管当年政策导向明朗，审核节奏加快 |
| 半年报披露前（7-8月） | 业绩窗口 | 大股东有动力在半年报前推出利好提振股价 |
| 年底（10-12月） | 国资考核窗口 | 国资大股东须在年末前完成当年资产证券化指标 |
| 重大政策出台后 | 政策响应窗口 | 如"并购六条"出台后，积压的重组项目集中推进 |

**内部紧迫节点（公司层面）：**

| 节点 | 判断方式 | 含义 |
|---|---|---|
| 大股东质押警戒线 | 当前股价 vs 质押价格×130%（典型警戒线） | 股价跌近警戒线时，重组救市的紧迫性极高 |
| 历史承诺到期日 | 查IPO承诺函中的时限约定 | 到期前若未履行承诺，大股东面临监管处罚 |
| 业绩承诺到期 | 上次并购后业绩承诺期通常为3年 | 承诺期结束后，原注入资产业绩可能大幅下滑，新一轮重组需求出现 |

---

### 5.3 等待成本评估

**核心问题：如果重组预期持续12个月未落地，持有成本是多少？**

```
等待成本 = 机会成本（无风险利率或其他投资回报）+ 股价自然回落风险

若重组预期泡沫化程度高（基础估值严重透支），等待成本 = 持有期回报 - 大盘收益

判断是否值得等待的简单公式：
期望收益（已在第七章计算）/ 预计等待时长（月）= 月化期望收益率

若月化期望收益率 < 1%，等待性价比极低，应降低仓位或转换标的。
```

## 六、第四层：重组方案质量评估（What & How Much）

> **核心目标：** 一旦方案草案/预案披露，快速判断"这是一个好重组还是坏重组"——注入的是真正有价值的资产，还是大股东借机转移利益。

---

### 6.1 注入资产质量深度分析

**怎么找资料：**
- 重组预案/草案：巨潮资讯下载，重点阅读"标的资产基本情况"和"盈利预测报告"
- 标的资产工商信息：天眼查/企查查查其近3年营收和利润变化趋势
- 行业研究报告：判断标的资产所在细分行业的竞争格局

**分析框架：**

**① 盈利能力真实性核查**

重组方往往有动机在注入前做高标的资产利润（以获得更高估值）。识别"突击利润"的方法：

```
警示信号：
- 注入资产注入前1-2年净利润增速显著高于行业均值（>50%）
- 应收账款增速远超营收增速（收入确认激进）
- 关联交易收入占比>30%（内部倒账）
- 现金流量净额与净利润差异超过30%（利润不含金量低）
- 资产评估机构选用与标的方关系密切的机构（利益冲突）
```

**② 竞争壁垒评估**

| 壁垒类型 | 评估问题 | 判断标准 |
|---|---|---|
| 技术壁垒 | 是否有核心专利？专利授权年限？ | 授权>5年且专利覆盖核心产品 |
| 客户壁垒 | 前五大客户是否稳定？是否为央国企/大型企业？ | 客户集中度高但客户质量好 |
| 牌照壁垒 | 是否有难以复制的行政许可？ | 存在稀缺牌照则溢价合理 |
| 规模壁垒 | 行业内市场份额排名？ | 行业前三名具有更强定价权 |

**③ 业绩承诺合理性检验**

业绩承诺通常为注入后3年的净利润目标。判断合理性：

```
历史净利润复合增速（CAGR）= (最近一年利润 / 三年前利润)^(1/3) - 1

承诺增速 = (第三年承诺利润 / 注入前一年利润)^(1/3) - 1

若承诺增速 > 历史CAGR × 1.5倍：承诺过于激进，未来减值风险高
若承诺增速 ≈ 历史CAGR：承诺合理
若承诺增速 < 历史CAGR：保守承诺，注入方有余地兑现
```

**业绩承诺未完成的后果**：补偿股份回购 → 上市公司承担商誉减值 → 股价下跌。所以激进的承诺期内股价有支撑，但承诺期结束后（第4年）往往是风险释放高峰。

---

### 6.2 定价公允性评估

**怎么找资料：**
- 重组预案：查"资产评估报告摘要"，获取评估方法（收益法/市场法）、增值率
- Wind/Choice：查同类上市公司（可比公司）当前PE/PB估值

**分析方法：**

**① 评估方法合理性**

| 评估方法 | 适用场景 | 潜在问题 |
|---|---|---|
| 收益法（DCF） | 盈利稳定的成熟企业 | 折现率和增长率假设主观性强，易被操纵 |
| 市场法（可比公司/交易） | 有充分市场参照的行业 | 市场高峰期使用会导致高估值 |
| 资产基础法 | 以资产为主的企业（房地产、金融） | 低估无形资产价值 |

**② 估值横向对比**

```
注入资产隐含PE = 评估价值 / 标的资产净利润

vs. 同类A股上市公司平均PE（Wind行业估值）
vs. 近期同类重组交易的平均PE（查证监会重组审核结果）

若注入资产隐含PE > 同类上市公司PE × 1.3：估值偏高
若注入资产隐含PE ≈ 同类上市公司PE：估值公允
若注入资产隐含PE < 同类上市公司PE × 0.8：注入资产估值低，上市公司股东实际占优
```

**③ 支付结构分析**

| 支付方式 | 信号含义 |
|---|---|
| 全股份支付（发行股份购买资产） | 注入方看好长期股价，愿意承担股价风险，利益高度绑定 |
| 现金 + 股份混合 | 注入方有短期兑现需求，部分套现 |
| 全现金支付 | 常见于产业并购，注入方无长期绑定 |

---

### 6.3 股份稀释效应完整测算

```
【摊薄后EPS测算模板】

基础数据：
  上市公司原股本 = A 亿股
  上市公司原净利润 = B 亿元
  注入资产净利润 = C 亿元（取承诺第一年）
  注入资产评估价值 = D 亿元
  定增发行价格 = E 元/股（通常为基准日前20日均价×90%）

计算过程：
  新增股份数量 = D / E（亿股）
  重组后总股本 = A + D/E（亿股）
  重组后归母净利润 = B + C（假设无协同效应）
  重组后EPS = (B + C) / (A + D/E)（元/股）

对比：
  重组前EPS = B / A
  EPS增厚比例 = (重组后EPS - 重组前EPS) / 重组前EPS × 100%

若EPS增厚 > 0：重组对原股东有利（增值并购）
若EPS增厚 < 0：重组稀释原股东利益（需质疑动机）
```

## 七、第五层：市场定价与安全边际（Price）

> **核心目标：** 量化"当前股价中到底定价了多少重组预期"，建立赔率/胜率框架，判断当前时点的性价比。

---

### 7.1 基础估值锚定（剥离重组预期）

**方法：假设重组完全不发生，当前股价合理吗？**

**怎么找资料：**
- 同花顺F10 / 亿牛网：获取历史PE/PB区间、行业估值中枢
- Wind行业估值：获取同行业平均PE
- 公司年报：获取基础业务净利润（剔除重组预期后的经常性利润）

**计算方法：**

```
【基础业务价值测算】

Step 1：确定基础业务净利润
  取最近12个月净利润，剔除非经常性损益（出售资产、政府补贴等）
  基础净利润 = TTM净利润 - 非经常性损益

Step 2：给予基础业务合理估值
  参考同行业可比公司PE中枢（景气高峰期打七折，景气低谷期溢价）
  基础业务合理市值 = 基础净利润 × 合理PE

Step 3：计算基础业务对应的股价
  基础股价 = 基础业务合理市值 / 总股本

Step 4：计算重组期权价值
  重组期权价值 = 当前股价 - 基础股价

若重组期权价值 > 0：市场已定价重组预期
若重组期权价值 / 当前股价 > 30%：重组预期占比过高，基础估值支撑弱
```

---

### 7.2 重组成功情景下的目标价测算

```
【重组成功情景目标价】

Step 1：测算重组后净利润
  重组后净利润 = 上市公司基础净利润 + 注入资产承诺净利润（取第一年）

Step 2：给予重组后合理PE
  参考：同行业重组后公司享受的平均PE溢价（通常重组后12-18个月享有较高估值）
  合理PE范围 = [行业均值PE × 1.0, 行业均值PE × 1.3]

Step 3：测算重组后合理市值
  重组后合理市值 = 重组后净利润 × 合理PE

Step 4：计算目标股价
  重组后总股本 = 原股本 + 新增股份（来自6.3测算）
  目标股价 = 重组后合理市值 / 重组后总股本

Step 5：测算潜在涨幅
  潜在涨幅 = (目标股价 - 当前股价) / 当前股价 × 100%
```

---

### 7.3 完整赔率/胜率期望收益框架

```
【三情景期望收益矩阵】

情景一：重组成功落地
  胜率估计（P₁）= ____%
  依据：动机评分 × 可行性分析 × 信号层级综合判断
  预期目标股价 = ____元（来自7.2测算）
  潜在收益（R₁）= +____%

情景二：重组失败，股价回落至基础估值
  概率（P₂）= ____%
  预期股价 = ____元（来自7.1基础估值）
  潜在损失（R₂）= -____%

情景三：重组悬而未决，长期横盘消耗
  概率（P₃）= 1 - P₁ - P₂
  预期收益（R₃）= ≈ 0%（或以机会成本计为负）

期望收益 = P₁ × R₁ + P₂ × R₂ + P₃ × R₃

决策标准：
  期望收益 > 15%：可介入
  期望收益 5%-15%：谨慎，考虑仓位
  期望收益 < 5%：性价比不足，放弃
```

### 7.4 仓位管理建议

| 信号层级（来自第五章） | 重组期权价值占股价比 | 建议仓位 |
|---|---|---|
| Level 1（弱信号） | <20% | 轻仓试探（≤3%仓位） |
| Level 2（中信号） | 20%-40% | 中等仓位（5%-8%仓位） |
| Level 3（强信号，未上会） | 30%-50% | 较重仓位（8%-12%仓位） |

| Level 4（已通过） | 高 | 评估是否兑现，关注解禁节奏 |

## 十、实操经验总结

**几条反直觉的经验：**

1. **"停牌筹划"≠ 利好**。停牌后重组失败复牌大跌是A股常见场景，停牌公告本身不是建仓信号。

2. **重组预期最强烈的时候往往是最危险的时候**。市场情绪最高涨时（各路资金都在讲故事时），往往预期已充分定价，安全边际最低。

3. **国资整合的确定性显著高于民企并购**。国资背景的整合有政策背书，审核也相对宽松，但溢价通常也已被市场定价。

4. **关注重组失败后的"二次机会"**。首次重组失败导致股价大跌后，若基础业务稳健且重组动机未消失，可能是更好的介入点。

5. **业绩承诺是双刃剑**。承诺期内股价有支撑，但承诺期结束后（通常第4年）往往是风险释放期，注意承诺到期时间节点。

6. **FA（财务顾问）的选择有信号意义**。顶级投行做FA，说明交易规模大且真实推进；小型机构做FA，可能只是走程序。

7. **问询函的质量比数量更重要**。一封问询函20个问题，说明监管关注度高、难点多；若回复后监管无进一步问询，通常意味着顺利推进。

## 十一、一页纸研究框架（快速筛选用）

```
【并购重组预期标的快速评估表】

■ 前置信息层（第一章）
□ 时间线整理完成，关键事件已标注信号层级
□ 无明显信息造假或内幕交易痕迹
□ 市场情绪温度：预期是否已过热？

■ 类型与动机（第二、三章）
□ 重组类型明确（国资整合/资产注入/借壳/产业并购）
□ 动机综合评分 ≥ 7/10

■ 可行性（第四章）
□ 资产负债表干净，商誉/总资产 < 20%
□ 定增间隔满足监管要求
□ 无ST/立案调查等监管障碍
□ 大股东持股 > 30%，控制力充足

■ 节奏与催化剂（第五章）
□ 当前处于 Level 2 或以上信号层级
□ 存在明确的时间窗口催化剂
□ 月化期望收益率 > 1%

■ 方案质量（第六章）
□ 注入资产无"突击利润"迹象
□ 业绩承诺增速不超过历史CAGR × 1.5倍
□ 注入资产隐含PE不高于同类上市公司 × 1.3倍
□ EPS测算为正增厚

■ 定价与赔率（第七章）
□ 基础股价测算：重组期权价值占当前股价比例合理
□ 三情景期望收益 > 15%

结论：
通过14项以上（含全部★★★★★风险项排查）：值得深入研究，可建仓
通过10-13项：谨慎关注，轻仓试探
通过9项以下：放弃，转移精力
```
