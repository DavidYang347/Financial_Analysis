## 一、商业模式与竞争优势

### 1.1 业务概述

| 字段 | 内容 |
|------|------|
| **主营业务** | 当前核心业务为**含锌铟固危废资源化利用**（精铟提取、锌锭生产），同步布局**高密度ITO靶材**制造。历史主业（土壤修复/生态水利）已大幅萎缩至营收5%以下。[[来源：公司2026年复牌公告]](https://paper.cnstock.com/html/2026-03/04/content_2184502.htm) |
| **收入来源构成** | 资源回收及综合利用业务：约95%（2025年前三季度）；精铟/粗铟/锌锭为主要产品；传统环保工程业务<5%。[[来源：东方财富财富号研报]](https://caifuhao.eastmoney.com/news/20260220130334058662370) |
| **地理收入拆分** | 以国内为主；云南省（云南业胜）为主要生产基地；已拟议在马来西亚布局（京蓝港槟），尚未产生收入。境外收入占比极低（未找到精确数据）。[[来源：公司公告2024]](https://file.finance.qq.com/finance/hs/pdf/2024/11/08/1221650187.PDF) |
| **核心客户群体** | B端为主，铟业务下游客户主要为**ITO靶材企业**，下游应用涵盖液晶显示、钙钛矿/异质结光伏电池、汽车调光玻璃。[[来源：公司2026年异动核查公告]](http://file.finance.sina.com.cn/211.154.219.97:9494/MRGG/CNSESZ_STOCK/2026/2026-3/2026-03-09/11985907.PDF) |
| **收入模式** | 资源回收业务：原料采购→冶炼提纯→产品销售（**一次性销售**为主）；未来ITO靶材：产品销售+可能的长期供货合同 |
| **价值链位置** | 当前处于产业链**中上游**（固危废原料→精铟提取）；正向下游延伸至ITO靶材（中游材料）。原料来源为锌铅冶炼厂固废，议价能力中等。 |
| **季节性特征** | 资源回收业务季节性不明显；铟价受全球供需、宏观经济及期货市场影响，波动性强；历史上环保PPP项目具有明显政府预算季节性，但该业务已大幅萎缩。 |
| **产品/服务生命周期** | 精铟提取处于**成长期**，需求受光伏/半导体驱动；ITO靶材业务处于**导入期**，尚未经过市场验证，面临6~12个月客户认证周期压力。[[来源：凤凰网财经]](https://finance.ifeng.com/c/8rDFzjRu1YZ) |
