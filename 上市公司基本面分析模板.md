## 九、风险识别清单

### 9.1 财务风险

- [ ] 现金流与利润长期背离，利润质量存疑
- [ ] 有息负债率过高，再融资压力大
- [ ] 应收账款异常增长，存在坏账风险
- [ ] 大额商誉面临减值压力
- [ ] 大股东股权高比例质押，存在爆仓风险
- [ ] 研发费用过度资本化，虚增利润
- [ ] 频繁进行非主业资产处置，依赖一次性收益维持利润
- [ ] 大额表外负债（担保/租赁/或有债务）未充分披露

### 9.2 经营风险

- [ ] 客户集中度过高（前五大客户占比 > 50%）
- [ ] 单一产品/业务依赖，多元化不足
- [ ] 核心技术人员/管理层离职风险
- [ ] 供应链集中，原材料/供应商依赖风险
- [ ] 产能利用率持续下滑，固定成本摊薄压力
- [ ] 产品/技术迭代滞后，被竞品侵蚀市场份额
- [ ] 海外业务依赖单一市场，地缘政治敏感度高

### 9.3 行业与外部风险

- [ ] 行业监管政策存在重大不确定性
- [ ] 竞争加剧导致毛利率持续下滑
- [ ] 技术路线存在被颠覆的风险
- [ ] 宏观经济下行对需求端的冲击
- [ ] 原材料/大宗商品价格波动超预期
- [ ] 国际贸易摩擦、出口管制、关税政策冲击

### 9.4 治理与合规风险

- [ ] 诉讼/仲裁重大事项悬而未决
- [ ] 实控人/创始人个人风险（健康、信用等）
- [ ] 股权结构变动导致经营不稳定
- [ ] 数据安全/隐私合规风险（互联网/医疗/金融等行业尤需关注）
