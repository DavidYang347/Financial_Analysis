## 一、商业模式与竞争优势

### 1.1 业务概述

| 字段 | 内容 |
| ------------- | ---------------------------------- |
| **主营业务** | 公司主要从事各类**输入类设备**（键盘、触控板、鼠标）、**柔性线路板（FPC）**及**钠离子电池**的设计、研发、制造和销售[[来源：东方财富百科]](https://baike.eastmoney.com/item/%E6%B1%9F%E8%8B%8F%E4%BC%A0%E8%89%BA%E7%A7%91%E6%8A%80%E8%82%A1%E4%BB%BD%E6%9C%89%E9%99%90%E5%85%AC%E5%8F%B8) |
| **收入来源构成** | 2025年：①笔记本/台式机电脑键盘等输入设备66.28%（14.22亿元）；②触控板/按键约~22%；③FPC约~8%；④钠离子电池<2%；⑤其他约1%[[来源：证券时报]](https://www.stcn.com/article/detail/3717263.html) |
| **地理收入拆分** | 2025年外销收入11.48亿元，占总营收53.52%；国内销售占46.48%；主要境外市场含日本、美国、韩国等[[来源：证券时报]](https://www.stcn.com/article/detail/3717263.html) |
| **核心客户群体** | 全球知名B端品牌商：ThinkPad、Lenovo联想、惠普HP、华为Huawei、三星Samsung、小米、富士康Foxconn等[[来源：东方财富百科]](https://baike.eastmoney.com/item/%E6%B1%9F%E8%8B%8F%E4%BC%A0%E8%89%BA%E7%A7%91%E6%8A%80%E8%82%A1%E4%BB%BD%E6%9C%89%E9%99%90%E5%85%AC%E5%8F%B8) |
| **收入模式** | 以**一次性产品销售**为主，接受客户定制化订单，依附大客户供应链体系运营 |
| **价值链位置** | 处于产业链**中游制造环节**：向上采购薄膜/导电材料、FPC基材、电池材料等；向下对接全球一线品牌整机厂；话语权偏弱，受客户集中度影响大 |
| **季节性特征** | 消费电子零组件业务受下游PC/笔记本市场节奏影响，Q3-Q4通常为旺季（圣诞备货），Q1相对淡季；钠离子电池业务初期季节性特征尚不明显 |
| **产品/服务生命周期** | 键盘/触控板处于**成熟期**但受AI PC迭代驱动升级；FPC处于**稳定成长期**；钠离子电池处于**产业化导入期**，商业化尚在早期摸索阶段[[来源：东方财富百科]](https://baike.eastmoney.com/item/%E6%B1%9F%E8%8B%8F%E4%BC%A0%E8%89%BA%E7%A7%91%E6%8A%80%E8%82%A1%E4%BB%BD%E6%9C%89%E9%99%90%E5%85%AC%E5%8F%B8) |

### 1.2 核心护城河评估

| 护城河类型 | 是否具备 | 评估说明 |
| ------- | ----- | -------------------- |
| 品牌壁垒 | ❌ | 消费者端无品牌；在B端客户中有一定认知度 |
| 成本优势 | ✅ | 规模化生产 + 一体化纵向整合（FPC自制）降低成本；全球键盘出货量约占20%以上，规模优势存在[[来源：东方财富百科]](https://baike.eastmoney.com/item/%E6%B1%9F%E8%8B%8F%E4%BC%A0%E8%89%BA%E7%A7%91%E6%8A%80%E8%82%A1%E4%BB%BD%E6%9C%89%E9%99%90%E5%85%AC%E5%8F%B8) |
| 网络效应 | ❌ | 制造业，无网络效应 |
| 转换成本 | ✅（弱） | 大客户认证周期长（通常1-2年），已通过认证的供应商切换成本较高 |
| 专利/资质壁垒 | ✅（一定） | 国家高新技术企业、专精特新"小巨人"；钠电领域有自主专利技术体系；行业标准参与情况未知 |
| 规模经济 | ✅ | 键盘年出货量约4000万套，触控板连续三年国内第一、全球前三；规模带来明显摊薄效应[[来源：东方财富百科]](https://baike.eastmoney.com/item/%E6%B1%9F%E8%8B%8F%E4%BC%A0%E8%89%BA%E7%A7%91%E6%8A%80%E8%82%A1%E4%BB%BD%E6%9C%89%E9%99%90%E5%85%AC%E5%8F%B8) |
| 数据壁垒 | ❌ | 制造业，无明显数据壁垒 |
| 生态系统绑定 | ❌ | 供应商体系，无生态绑定 |
| 地理/渠道垄断 | ❌ | 存在竞争对手（达方、群光等），无垄断地位 |

> **护城河宽度评级**：窄护城河
>
> **分析说明**：传艺在键盘、触控板细分市场具备相当规模优势和一定的客户黏性（认证壁垒），但整体护城河较窄。公司主营消费电子代工制造，产品差异化程度有限，议价能力受制于品牌客户。钠离子电池作为新业务，尚处于产业化早期，护城河尚未建立。

### 1.3 行业格局（波特五力分析）

| 字段 | 内容 |
| ---------- | ----------------------- |
| **行业生命周期** | 消费电子零组件：成熟期；钠离子电池：导入期 |
| **市场集中度** | 键盘市场：头部企业包括达方电子（群创）、精元、传艺科技、富士康旗下工厂等，CR3约60-70%（估算）；FPC市场竞争较分散；具体CR3/CR5数据未找到相关信息 |
| **主要竞争对手** | ①**达方电子（台湾/苏州）**：键盘和笔记本配件老牌厂商，产品覆盖范围广，深耕日韩客户；②**精元电脑（台湾/常熟）**：键盘领域规模较大，主供联想等；③**鹏鼎控股（002938）**：FPC领域巨头，规模远超传艺[[来源：深交所问询函回复]](https://static.cninfo.com.cn/finalpage/2023-07-01/1217184993.PDF) |
| **潜在替代威胁** | 键盘：语音输入/AR手势交互长期存在替代可能，但短期内实体键盘需求稳定；钠电：竞争激烈，锂电价格下行形成直接竞争 |
| **公司市场份额** | 笔记本键盘全球占比约20%+，国内领先；触控板国内第一、全球前三；FPC市场份额较小[[来源：东方财富百科]](https://baike.eastmoney.com/item/%E6%B1%9F%E8%8B%8F%E4%BC%A0%E8%89%BA%E7%A7%91%E6%8A%80%E8%82%A1%E4%BB%BD%E6%9C%89%E9%99%90%E5%85%AC%E5%8F%B8) |

**波特五力量化评估**：

| 竞争力量 | 强度 | 关键判断依据 |
| ---------- | ----- | --------------------- |
| 现有竞争者的竞争强度 | 高 | 键盘/FPC行业竞争激烈，价格战持续，毛利率呈下行趋势（2022:24.28%→2024:21.41%）[[来源：新浪财经鹰眼]](https://finance.sina.com.cn/stock/yyyj/2025-04-29/doc-ineuufeh4764900.shtml) |
| 新进入者威胁 | 中 | 规模门槛和客户认证构成一定壁垒，但资本需求不算极高 |
| 替代品威胁 | 中 | 短期PC键盘无成熟替代方案；钠电面临锂电等多路替代 |
| 供应商议价能力 | 中 | 主要原材料为薄膜材料、FPC基板、电池材料，来源多元，但大宗商品价格波动有影响 |
| 买方议价能力 | 高 | 主要客户为联想、惠普等全球巨头，议价能力极强；客户集中度高进一步加剧买方强势 |

> **行业吸引力综合评估**：消费电子零组件制造行业整体吸引力中等偏弱——竞争格局激烈、买方强势、规模化优势逐步趋于平衡。传艺的相对优势在于细分赛道（键盘/触控板）的规模领先地位，但毛利率长期面临下行压力。钠离子电池赛道潜力较大但当前变现能力有限、亏损拖累明显。

### 12.2 投资决策清单

**商业模式与护城河**
- 公司靠消费电子零组件代工赚钱，核心逻辑清晰但护城河有限
- 键盘/触控板市场份额领先，未来5年份额难以大幅被侵蚀，护城河基本可维持
- 钠电领域宁德时代等巨头入场，传艺的先发优势将面临较大侵蚀风险

**财务质量**
- 2025年经营CF/净利润约2.97倍，现金质量良好✅
- ROE 2025年约4.2%，仍低于15%；历史上长期摧毁价值❌
- 过去3年未出现财务重述，审计意见为标准无保留意见✅

**估值合理性**
- 当前80倍PE对消费电子代工而言明显偏高，钠电期权价值是溢价核心
- 若核心假设下调20%（净利润仅6900万），市场当前价格安全边际不足
- 相对SOTP估值，当前价格高出中性估值约15-25%

**管理层与治理**
- 消费电子主业执行力较好，过去承诺兑现率尚可✅
- 大股东扭亏后减持，与公开积极表态不完全一致❌
- 大股东质押比例7.9%，在安全范围内✅

**风险与安全边际**
- 最悲观情景（钠电二次减值+关税冲击）：股价可能回撤至13-15元，下行风险约35-40%，较难接受
- 最重要"一票否决"风险：美国进一步对华电子产品加征关税且越南工厂产能不足以覆盖

### 12.3 核心投资逻辑

> 传艺科技的投资逻辑是：**以AI PC换机周期驱动的消费电子主业稳健增长为底仓基础，以钠离子电池商业化落地为弹性期权**。主业在AI PC换机红利下有稳定的收入增速（约10%/年），越南工厂对冲关税风险保住份额，经营现金流已在2025年实质性改善；如果钠电在2026-2027年实现规模化出货（特别是在电动二轮车市场），将驱动公司营收和净利润大幅跨越，届时估值可能再次重估。但这一逻辑成立的前提是钠电能够实质性突破，当前并无确定性。

**逻辑成立的关键前提**：

1. 消费电子主业维持年均10%营收增长，AI PC换机周期持续到2027年
2. 越南工厂顺利爬坡并承接足够的海外订单（包括潜在新客户如LG）
3. 钠离子电池在电动二轮车或储能领域获得至少一个批量化大客户，2026-2027年实现盈亏平衡
