## 一、商业模式与竞争优势

### 1.1 业务概述


| 字段            | 内容                                                                                                                             |
| ------------- | ------------------------------------------------------------------------------------------------------------------------------ |
| **主营业务**      | 专注光伏发电领域，从事**组串式逆变器**（并网逆变器+储能逆变器）的研发、生产、销售和服务；同时通过子公司锦浪智慧开展**分布式光伏电站**的开发、投资、建设及运营。公司定位为"用技术的力量推动清洁能源成为全球主力能源"。               |
| **收入来源构成**    | 光伏逆变器（含配套产品）约**68.05%**（2024年44.52亿元）；分布式光伏发电（电站发电+户用系统服务）约**31.95%**（2024年约20.9亿元）。逆变器为核心，电站业务为新增长极。                          |
| **地理收入拆分**    | 境外收入占比较高，约**60-70%**（欧洲、亚非拉、美洲为主要海外市场；欧洲以户用逆变器+储能为主，亚非拉以并网逆变器为主）。国内收入约30-40%，毛利率显著低于海外（国内不足10%）。产品销往100余个国家地区。                 |
| **核心客户群体**    | 主要面向**B端**（分销商/经销商为主要渠道，大型能源企业、EPC工程商等）；分布式光伏电站业务含工商业客户（"自发自用、余电上网"）和居民客户（"全额上网"）。                                             |
| **收入模式**      | 逆变器业务：**一次性产品销售**（标准品，海外溢价显著）；分布式电站业务：**持续性电力销售收入**（工商业电站长期售电）+**服务收入**（户用系统安装运维）；部分电站可通过出售资产变现。                               |
| **价值链位置**     | 处于光伏产业链**中游**（光伏系统核心设备环节）。上游为IGBT、电容、变压器等电子元器件供应商，下游为经销商/分销商和终端电站业主。在分布式市场具备较强话语权，逆变器在系统成本中占比约5-10%但技术壁垒高，具有一定定价权。             |
| **季节性特征**     | 逆变器业务具有明显季节性：**Q2、Q3为旺季**（夏季日照充足，全球装机节奏），Q1、Q4为淡季。分布式发电收入也受季节影响（夏季发电量高于冬季）。2024年数据印证：Q2净利润3.32亿、Q3净利润3.17亿，而Q1仅0.20亿、Q4仅0.22亿。 |
| **产品/服务生命周期** | 组串式逆变器处于**成长期**（全球分布式光伏高速渗透）；储能逆变器处于**导入-成长期**（户储市场欧洲2023年去库后正复苏，工商业储能快速崛起）。公司已从第一代迭代至第六代产品，当前最高转换效率99.1%。                     |

| 护城河类型   | 是否具备  | 评估说明                                                                 |
| ------- | ----- | -------------------------------------------------------------------- |
| 品牌壁垒    | ✅     | 产品销往100余个国家，彭博全球逆变器可融资性排名亚洲前三，全球第一家获PVEL可靠性测试报告的逆变器企业，在分布式市场中品牌认知度较强 |

| 网络效应    | ❌     | 逆变器为硬件产品，不具备典型网络效应，云监控平台（锦浪云）用户粘性有限                                  |
| 转换成本    | ✅（中等） | 逆变器与光伏组件/储能电池的适配性、调试服务绑定、运维数据积累形成一定转换成本；但行业内更换品牌的成本并非极高              |
| 专利/资质壁垒 | ✅     | 截至2025年9月末累计拥有专利334项；持续被认定为国家高新技术企业；与上交大合作建有院士工作站；持有多项国际认证（SAA、ETL等） |
| 规模经济    | ✅     | 2024年12月末95万台组串式逆变器项目建成，产能进一步提升；规模扩大有助于分摊固定成本                        |
| 数据壁垒    | ✅（初步） | 多年运营分布式电站积累发电数据，锦浪云平台积累客户数据，但尚未形成显著竞争壁垒                              |
| 生态系统绑定  | ❌（弱）  | 目前尚未形成强有力的生态绑定，储能系统业务起步，生态建设仍在初期                                     |
| 地理/渠道垄断 | ❌     | 无特定区域独占资源，依赖全球分销商网络                                                  |


> **护城河宽度评级**：**一般（偏窄）**
>
> **分析说明**：锦浪科技在分布式光伏组串逆变器细分领域拥有相对明确的品牌和技术积累，全球市占率约8%（第三位），但整体护城河宽度有限。行业竞争激烈、产品同质化趋势明显，华为和阳光电源压制效果显著，第二梯队竞争白热化（价格战），护城河面临侵蚀风险，尤其在国内市场。分布式电站业务的现金流稳定性是重要的护城河补充，但负债较高。

### 1.3 行业格局（波特五力分析）


| 字段         | 内容                                                                                                                  |
| ---------- | ------------------------------------------------------------------------------------------------------------------- |
| **行业生命周期** | **成长期**（全球光伏装机高速增长，2024年中国新增277.57GW同比+28.33%；全球2028年预计668-1112GW新增装机）                                              |
| **市场集中度**  | 全球CR2（华为+阳光电源）约46.7%，CR5约71%（2022年数据，中国企业均入前五）；国内大型项目中华为+阳光电源合计中标占比超28%，TOP10占比56%。集中度高但第二梯队竞争激烈。                   |
| **主要竞争对手** | **阳光电源**（全球No.1，技术全面，大型地面电站+储能布局完整）；**华为**（全球No.2，技术生态能力强，智能光储）；**固德威**（与锦浪定位相近，分布式光储，同处第二梯队）；**古瑞瓦特**（全球前五，海外布局积极） |
| **潜在替代威胁** | 微型逆变器（禾迈股份等）在户用屋顶场景与组串式形成一定竞争；家电企业（奥克斯、海尔、美的）跨界进入；长期来看储能一体化可能改变逆变器单独销售模式。替代威胁**中等**。                                |
| **公司市场份额** | 全球约**8%**（2022年排名第三），近年来受欧洲去库+国内价格战影响份额有所承压，趋势**稳定偏下**。                                                             |


**波特五力量化评估**：


| 竞争力量       | 强度     | 关键判断依据                                                                                           |
| ---------- | ------ | ------------------------------------------------------------------------------------------------ |
| 现有竞争者的竞争强度 | **高**  | 2024年国内逆变器中标价已跌至0.1元/W甚至0.081元/W（历史高点下降50%以上）；150余家企业中标，价格战激烈；毛利率连续三年下滑（2022年33.52%→2024年31.57%） |
| 新进入者威胁     | **中**  | 家电企业、光伏组件企业跨界进入；但逆变器技术壁垒（功率电子、算法、认证）使非专业玩家难以快速追赶；核心市场对品牌可靠性要求较高                                  |
| 替代品威胁      | **中**  | 微型逆变器渗透率提升，在某些应用场景具优势；储能一体化或逐步改变独立销售模式                                                           |
| 供应商议价能力    | **中**  | IGBT等核心元器件曾出现供应紧张；但随行业规模扩大和国产替代推进，供应商集中度风险有所下降；公司前五大供应商采购额占比约为21%                                |
| 买方议价能力     | **中高** | 分销商议价能力随行业竞争加剧而提升；国内大型电站项目业主集中度高，批量采购压价；海外市场分散程度较高，单一买方影响有限                                      |


> **行业吸引力综合评估**：光伏逆变器行业受益于全球能源转型，长期成长空间可观，但当前处于行业内卷加剧阶段，竞争烈度高、价格持续下行。锦浪科技凭借分布式市场领先地位和海外布局具有相对优势，但绝对优势相比龙头尚不显著。分布式光伏发电业务提供的稳定现金流使公司在逆变器行业波动中具备一定抗周期能力。

### 11.4 当前核心看点总结

**看点一：储能系统业务从0到1的放量验证**

**看点二：欧洲市场复苏与英国新政策带来的逆变器出货量修复**

> 欧洲户储市场2023-2024年经历深度去库，2025年下半年起逐步修复。英国《温暖家园计划》（2026年启动）是重要新增政策利好。锦浪在英国市场占比约10-20%，若英国装机提速，将直接带动公司海外高利润市场的出货量。欧洲业务的高毛利属性（远高于国内和亚非拉）使得欧洲复苏对净利润的弹性贡献显著。

**看点三：分布式光伏电站业务现金流价值重估**

> 公司累计并网装机容量已达1,360.98MW，分布式电站业务毛利率约60%，贡献稳定高质量现金流。随着电站投资放缓（CAPEX显著下降）和存量电站产生的稳定发电收益增加，这部分"隐性价值"有望逐步被市场认识——尤其是可转债募资后公司可能进一步扩大电站组合规模，或通过电站出售变现创造额外收益。

### 12.3 核心投资逻辑

> 锦浪科技是全球分布式光伏组串式逆变器市场的重要参与者，当前正经历从"逆变器设备商"向"光储一体化解决方案提供商"的战略转型关键期。核心投资逻辑在于：①全球分布式光伏装机长期向好，公司作为全球前三大逆变器厂商享有市场成长红利；②2026年储能系统业务正式发货，新成长曲线有望打开，打破市场对公司"逆变器单一业务"的固有认知；③高毛利分布式光伏电站资产（约1361MW）提供稳定且可观的现金流支撑，为公司穿越行业周期提供缓冲；④欧洲户储复苏+英国政策红利驱动出货量改善，业绩有望在2025/2026年实现显著修复。

**逻辑成立的关键前提**：

1. 储能系统业务2026年能够实质放量（收入>3亿元）且毛利率达到预期水平

3. 国内逆变器价格战不继续大幅恶化（毛利率在30%以上）
4. 资产负债率持续改善，流动性压力得到缓解（可转债融资落地+电站现金流回收）
