## 1. 公司做什么

### 1.1 业务结构

| 维度 | 内容 |
| --- | --- |
| **主营业务** | 高分子材料抗老化助剂（抗氧化剂、光稳定剂、U-pack）研发、生产、销售；同时布局润滑油添加剂、生命科学、电子级PI材料三大新业务曲线 [[来源：公司官网]](https://www.rianlon.com) |
| **收入构成** | 抗老化添加剂2025年收入47.74亿（79.3%）、润滑油添加剂12.07亿（20.0%）、其他0.42亿（0.7%）；其中抗氧化剂19.20亿、光稳定剂20.75亿、U-pack 5.96亿 [[来源：东兴证券研报]](https://www.hangyan.co/reports/3885917334207989507) |
| **客户结构** | B端为主；客户覆盖BASF、DSM、杜邦、朗盛、PPG、金发科技、万华化学等全球高分子材料龙头；累计国内外客户3000余家；润滑油客户含中石油、中石化及润英联等国际四大添加剂公司 [[来源：证券之星]](https://stock.quote.stockstar.com/corp_300596.shtml) |
| **收入模式** | 产品销售为主（B2B大宗化学品）；高端客户采用U-pack一体化解决方案，粘性更高；长期框架合同+短期订单结合 |
| **区域结构** | 国内外并重；国内设6大生产基地（天津汉沽、宁夏中卫、浙江常山、河北衡水、广东珠海、内蒙古赤峰）；海外在德国、美国、香港、日本设子公司，在欧美亚设仓库，服务全球客户 [[来源：证券之星]](https://stock.quote.stockstar.com/corp_300596.shtml) |
| **业务阶段** | 核心业务（抗老化助剂）处于成熟稳定增长期；润滑油添加剂处于快速成长期；生命科学、PI材料处于导入/培育期 |
| **季节性** | 化工品季节性不显著；Q1因春节存在一定需求淡季，整体受下游行业开工率影响较大 |

## 2. 护城河与行业格局

### 2.1 护城河

| 护城河类型 | 是否具备 | 说明 |
| --- | --- | --- |
| 品牌 | 是 | "利安隆"品牌在全球高分子材料助剂领域有较高知名度，目标"百年利安隆"，B端客户信任度高 [[来源：公司官网]](https://www.rianlon.cn/resources-information/news/173) |
| 成本优势 | 是 | 规模效应带来原材料集中采购优势；6个国内基地+持续技改降本；但行业竞争加剧下成本优势被压缩 |
| 技术/专利 | 是 | 拥有主抗氧剂、辅抗氧剂、UVA、HALS全系列生产核心技术；研发院下设工艺工程和应用技术两大研究所；持续推出差异化新品 [[来源：中财网]](https://quote.cfi.cn/gsda/19557/300596.html) |
| 规模效应 | 是 | 国内抗老化助剂产能24.21万吨（2025年），居国内第一；规模优势支撑单位固定成本摊薄 [[来源：东兴证券研报]](https://www.hangyan.co/reports/3885917334207989507) |
| 转换成本 | 是（中） | 大客户供应链认证周期长（通常6-18个月），客户轻易不换供应商；U-pack产品粘性更强，形成定制化壁垒 |
| 渠道/资源 | 是 | 全球服务网络（欧美亚多个仓库，72小时配送）；20年海外客户积累；与润英联等国际巨头建立深度战略合作 [[来源：利安隆官网]](https://www.rianlon.cn/resources-information/news/173) |
| 数据/生态 | 否 | 暂无显著的数据或生态壁垒 |

> **护城河结论**：`一般偏宽`
>
> **关键判断**：护城河真实存在——在品牌、技术、规模、渠道方面均有积累，"全系列产品配套"是最核心的差异化壁垒（全球仅两家）。但护城河正面临挑战：2023年以来行业竞争加剧（国内产能过剩、落后产能出清），价格承压明显；主业毛利率从2018年31%降至2025年22%，盈利能力持续受侵蚀，说明定价权有所减弱。护城河可持续性依赖于技术创新和高端产品比例提升。

### 2.2 行业格局

| 维度 | 内容 |
| --- | --- |
| **行业生命周期** | 抗老化助剂整体处于成熟期，需求随高分子材料消费稳定增长（全球4-5%/年，国内5-8%/年）；新能源汽车、光伏、电子等新兴领域带来增量需求 [[来源：天风证券研报]](https://pdf.dfcfw.com/pdf/H3_AP201907151338687454_1.pdf) |
| **竞争格局** | 全球市场：BASF、韩国Songwon、SI Group、ADEKA合计约占42-85%（HALS/UVA类）市场份额，国内企业整体份额低但持续提升；国内市场相对分散但已向龙头集中趋势明显，利安隆是国内龙头，2023年后落后产能加速出清，订单向行业前几名集中 [[来源：观研报告网]](https://m.chinabaogao.com/tuozi/202407/715261.html) |
| **主要竞品** | ①风光股份（301100，国内抗氧化剂标准主导者，产品偏单一）；②宿迁联盛（HALS全球约17%份额，非上市）；③鼎际得（抗氧化剂专注型）；利安隆相对优势在于产品系列完整度和U-pack解决方案能力；海外巨头BASF差距仍大 [[来源：国海证券研报]](https://www.nxny.com/report/view_5166718.html) |
| **替代风险** | 低——化学助剂是功能性必需品，目前尚无可替代的技术路线；但生物基材料内建稳定性是潜在长期替代方向（影响周期10年+）|
| **供需状态** | 国内产能偏过剩，行业竞争激烈，价格承压；但2023年后市场自然出清正在进行，利安隆等龙头份额有望持续提升 |
| **价格趋势** | 产品价格整体承压下行（2022年高点至今已明显下跌，出货量增长但营收增速低于出货量增速）；毛利率改善主要靠降本和结构升级，而非提价 [[来源：小牛行研]](https://www.hangyan.co/reports/3637318979342567167) |

> **行业结论**：行业整体是"逆风+筑底"格局——供需失衡背景下，竞争激烈、价格承压，但行业出清正在进行，龙头企业相对受益；新能源汽车、光伏等新兴应用方向是增量市场，利润池有望在出清完成后重新扩大。

### 2.3 定价权

| 维度 | 判断 |
| --- | --- |
| **提价能力** | 弱——历史上曾具备一定提价能力，但2023年以来行业竞争加剧，公司以量换价，出货量增速（+11.63%）明显高于收入增速（+3.92%） |
| **顺价能力** | 中——原材料成本波动时有一定传导能力，但传导滞后且不完整；近两年靠降本（供应链优化、技改）而非提价来维护毛利率 |
| **客户议价** | 较强——大客户（BASF等跨国巨头）议价能力强，但通过U-pack定制化方案和全球服务网络可以部分对冲 |
| **竞争压力** | 高——国内同业价格竞争激烈，落后产能虽在出清但过程漫长 |

> **定价权结论**：`弱-中`，正处于行业竞争最激烈阶段，定价权受压，但龙头优势使其比同行更能维护毛利率。

| **估值预期差** | 市场按传统化工企业估值（低PE）定价 | 若新材料（PI、生命科学）业务规模化，可能切换至新材料/科技制造估值框架（更高PE或PS） | 业务多元化战略推进，收入结构向高端材料迁移 | 存在估值体系重估可能 |

> **最大预期差**：**行业出清节奏和毛利率修复弹性**——市场普遍低估了竞争格局改善的速度，利安隆出货量以远超行业增速的节奏扩张（量增11%+ vs 行业4-5%），意味着落后产能正在快速萎缩。一旦价格止跌回升，毛利率可能从22%快速修复至25%+，利润弹性可能使2026-2027年实际利润超一致预期20-30%。市场可能在2026年Q2-Q3的季报中意识到这一预期差。

### 8.10 一段话总结

> 利安隆是国内高分子材料抗老化助剂的绝对龙头（全球仅两家全系列供应商之一），依托20年技术积累和全球客户网络，正在行业竞争出清中逐步扩大市场份额。核心逻辑是"量增+毛利率修复+多元新业务赋能"，2025年营收60亿、净利润5亿，连续14年复合增速24%。**最大预期差在于市场低估了行业出清速度和毛利率修复弹性**——若抗老化助剂价格在2026年企稳，利润弹性可超预期20-30%。未来3年核心催化剂是：润英联大客户深化（润滑油增速维持15%+）、PI材料国产替代商业化突破（2026年）、马来西亚产能出海（2027年投产开启新成长周期）。**主要风险是高PE估值（历史97分位）叠加有息负债上升和应收账款压力，安全边际有限**，建议在股价回调至40-43元（对应2026E PE 15-16x）区间分批布局，或在季报毛利率超23%的右侧信号出现后加仓；当前45元附近可少量参与，但非加仓最优时机。
