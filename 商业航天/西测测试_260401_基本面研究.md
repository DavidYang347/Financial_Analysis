## 一、商业模式与竞争优势

### 1.1 业务概述

| 字段 | 内容 |

| **收入来源构成** | 2024年：环境与可靠性试验45.97%、电装业务37.03%、电磁兼容性试验10.42%、电子元器件检测筛选5.46%、检测设备销售0.56%、其他0.57%。[[来源：新浪财经2024年报解读]](https://finance.sina.com.cn/stock/aigc/stockfs/2025-04-28/doc-ineusnuy9211245.shtml) |
| **地理收入拆分** | 收入基本来自中国大陆，无境外收入。主要市场集中于西安、成都、北京等地[[来源：2022年报]](https://stockn.xueqiu.com/SZ301306/20230425851385.pdf)；未找到精确地区拆分数据。 |
| **核心客户群体** | 主要为军工集团下属子公司及科研院所（如航空工业、中国航天、中国电科等），及民用飞机领域（中国商飞）。B端政企客户为主。[[来源：2024年报摘要]](https://epaper.stcn.com/pic/202504/28/1b30684cf330f456abdc4737177b72bc.pdf) |

| **季节性特征** | 有明显季节性。军工客户采购计划与结算集中在下半年，特别是Q4，公司四季度收入及现金流通常显著高于前三季度。[[来源：2022年报]](https://stockn.xueqiu.com/SZ301306/20230425851385.pdf) |
