## 一、商业模式与竞争优势

### 1.1 业务概述


| 字段            | 内容                                                                                                                                                                                  |
| ------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **主营业务**      | 三大业务：①**锂电材料**（电池级氢氧化锂、电池级碳酸锂），是锂离子动力电池正极材料的核心原料；②**防静电超净技术产品**，服务半导体、新型显示等高端制造业；③**医疗器械**，一次性使用无菌医疗器械                                                                              |
| **收入来源构成**    | 锂电材料86.08%（64.98亿元）、防静电超净技术10.27%（7.75亿元）、医疗器械3.65%（2.75亿元），2025年全年数据[^6]([天华新能2025年报解读-新浪财经](https://finance.sina.com.cn/stock/aigc/stockfs/2026-03-19/doc-inhrpsps4596251.shtml)) |
| **地理收入拆分**    | 中国大陆96.87%（73.13亿元，+35.64%），海外3.13%（2.36亿元，-80.59%）；海外业务大幅萎缩，国内成增长主力[^6]([天华新能2025年报解读-新浪财经](https://finance.sina.com.cn/stock/aigc/stockfs/2026-03-19/doc-inhrpsps4596251.shtml))  |
| **核心客户群体**    | B端为主：新能源汽车动力电池厂商（含宁德时代）、消费电子电池厂商、储能电池厂商；防静电产品客户包含AMD、美光、中芯国际、京东方等[^7]([天华新能公司资料-同花顺](https://basic.10jqka.com.cn/mobile/300390/company.html))                                       |
| **收入模式**      | 一次性产品销售为主；锂电材料兼有来料加工模式（与宁德时代合作，收取稳定加工费）[^8]([天华新能推动质量回报双提升-新浪财经](https://finance.sina.com.cn/stock/relnews/cn/2024-03-03/doc-inakznhn6246578.shtml))                                |
| **价值链位置**     | **中游**：处于锂矿资源开采（上游）和电池正极材料/动力电池制造（下游）之间，为锂盐冶炼加工商；公司正向上延伸布局锂矿资源，逐步向"资源+冶炼"一体化转型[^9]([天华新能锂资源布局-东方财富](https://caifuhao.eastmoney.com/news/20251210165431931337850))                   |
| **季节性特征**     | 季节性不明显，但受锂价波动影响较大；行业存在补库/去库周期带来的阶段性需求波动                                                                                                                                             |
| **产品/服务生命周期** | 锂电材料处于成长期（受益新能源渗透率提升，但当前处于行业供给过剩周期）；防静电超净技术处于成长期；医疗器械处于成熟期                                                                                                                          |

> **护城河宽度评级**：一般
>
> **分析说明**：天华新能的核心护城河在于产能规模、客户认证壁垒以及与宁德时代的战略绑定关系。公司在氢氧化锂领域属全球前三，具备一定先发优势。但锂盐行业整体面临同质化竞争加剧、产能过剩等问题，护城河面临侵蚀风险。宁德时代成为第二大股东后，绑定关系进一步加深，一定程度上提升了客户稳定性，但也带来大客户集中风险。

### 1.3 行业格局（波特五力分析）


| 字段         | 内容                                                                                                                                                                |
| ---------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **行业生命周期** | 成长期（新能源长周期成长趋势不变，但当前处于行业阶段性供给过剩周期）                                                                                                                                |
| **市场集中度**  | 全球前三大厂商（赣锋锂业、雅宝、天华新能）合计占全球约52%市场份额[^11]([全球电池级氢氧化锂市场-恒州诚思](https://www.yhresearch.cn/reports/2153613/lithium-hydroxide-for-battery))；国内格局较分散，两超多强                 |
| **主要竞争对手** | 赣锋锂业（综合实力强，上下游一体化更完整）、天齐锂业（锂矿资源优势显著）、雅宝（全球化布局强）、盛新锂能、中矿资源、容汇锂业等[^11]([全球电池级氢氧化锂市场-恒州诚思](https://www.yhresearch.cn/reports/2153613/lithium-hydroxide-for-battery)) |
| **潜在替代威胁** | 磷酸铁锂（LFP）电池扩张对高镍三元（需大量氢氧化锂）路线有一定替代压力；固态电池技术成熟后对液态电池材料体系有长期替代风险                                                                                                    |
| **公司市场份额** | 全球约17-18%（估算，位居前三），近3年趋势：受锂价下跌量增价减，市场份额保持相对稳定                                                                                                                     |


**波特五力量化评估**：


| 竞争力量       | 强度  | 关键判断依据                                                  |
| ---------- | --- | ------------------------------------------------------- |
| 现有竞争者的竞争强度 | 高   | 产能过剩导致价格战激烈，锂盐价格2022年峰值超50万元/吨，2024年已跌至7-11万元/吨低位       |
| 新进入者威胁     | 中   | 资本门槛较高，但大量玩家已入场，新增产能持续释放                                |
| 替代品威胁      | 中   | LFP路线扩张对高镍三元路线有压力，但长期来看高能量密度需求仍支撑氢氧化锂需求                 |
| 供应商议价能力    | 高   | 锂精矿供应商集中（主要在澳大利亚、南美），公司当前自给率有限，受锂矿价格波动影响大；公司正通过海外锂矿布局改善 |
| 买方议价能力     | 高   | 宁德时代等大型客户采购量大，且有多元供应商选择权，定价话语权较强                        |


> **行业吸引力综合评估**：当前行业处于供给过剩周期，竞争激烈，短期盈利能力承压。但中长期来看，随着新能源汽车及储能需求持续增长，锂盐需求将保持成长趋势，行业长期空间广阔。天华新能凭借规模优势和宁德时代战略绑定，在竞争中具备一定相对优势，但需密切跟踪供需格局改善时间节点。

### 1.4 定价权分析

- 核心技术人员/管理层离职风险（新任总裁较新，关注过渡风险）

- 产品/技术迭代滞后风险（固态电池布局已启动，但商业化较远）

### 9.3 行业与外部风险

- 行业监管政策存在重大不确定性

- **宏观经济下行**：强周期行业，新能源汽车需求若不及预期将直接冲击业绩
- **原材料价格波动**：锂精矿价格波动剧烈，成本控制难度大
- **国际贸易风险**：海外锂矿政治风险、关税政策不确定性

### 11.4 当前核心看点总结

**看点二：锂价周期底部的反转弹性**

> 天华新能具有极高的业绩弹性：2022年锂价高峰时毛利率64.44%、净利润65.86亿，2025年锂价低迷时净利润4.02亿，两者相差16倍。一旦锂价从当前7-10万元/吨区间回升至15-20万元/吨，公司利润将出现爆发式增长。当前股价在一定程度上已反映了对锂价反转的期待，关键是反转时间节点。

**看点三：港股上市与资源一体化布局**

> 2026年3月公告港股上市计划，将为公司开辟新的国际融资渠道，有助于加速海外锂矿资源开发（已布局尼日利亚、刚果金、国内江西/四川等锂矿资源）。若资源一体化成功，将在下一个锂价上行周期中大幅提升公司成本竞争力，从"纯加工"转型为"资源+加工"一体化龙头。

### 12.3 核心投资逻辑

> 天华新能的核心投资逻辑是：作为全球头部氢氧化锂供应商，公司当前正处于行业周期底部，但凭借宁德时代战略入股的稳定客户关系、大幅扩张的产能规模（16.5万吨/年）和积极布局的锂矿资源一体化，一旦锂盐价格企稳回升，公司将享有极高的利润弹性（参考2022年净利润65.86亿的历史高峰），股价有望实现大幅修复。同时，防静电超净技术业务高增长（+35.95%）和港股上市提供中期催化。

**逻辑成立的关键前提**：

1. 锂盐价格在未来12-24个月出现显著回升（至少回升至15万元/吨以上）
2. 宁德时代战略合作持续深化，来料加工量保持稳定或增长
3. 经营活动现金流恢复为正，应收账款回款情况良好，财务健康度改善
