## 2. 护城河与行业格局

### 2.1 护城河

| 护城河类型 | 是否具备 | 说明 |
| --- | --- | --- |
| 品牌 | 是 | 依托中国有研品牌，在军工领域口碑稳固；民品中成为头部手机厂商供应商 |
| 成本优势 | 一般 | 规模尚小（2024年营收6亿），成本优势有限；但粉末冶金产线建成国内第一条400吨线 |
| 技术/专利 | 是 | 9项核心技术达国际领先或先进，多项专利及独家技术（航空锻件、硅铝封装）；但部分专利来自继承有研总院 [[来源：IPO问询]](https://sdsyw.daiweicd.com/post/11556.html) |
| 规模效应 | 否 | 当前营收体量较小，规模效应弱 |
| 转换成本 | 是（军品） | 军品装备型号深度绑定，换供应商成本极高；民品转换成本较低 |
| 渠道/资源 | 是 | 中国有研集团资源（央企渠道）、战略股东（国华卫星、中航产业基金、比亚迪）深度绑定 [[来源：招股说明书]](http://money.finance.sina.com.cn/corp/view/vCB_AllBulletinDetail.php?id=11801637&stockid=688811) |
| 数据/生态 | 否 | 无明显数据/生态优势 |

> **护城河结论**：`一般偏窄`
>
> **关键判断**：`军品端护城河较宽（独家供方+资质壁垒），但民品端（智能终端/汽车）竞争格局快速恶化；技术护城河真实但需持续投入维护，有被专精特新竞争者追赶的风险。`

### 2.2 行业格局

| 维度 | 内容 |
| --- | --- |
| **行业生命周期** | 成长期（金属复合材料国产化替代+民用渗透）[[来源：华金证券研报]](https://pdf.dfcfw.com/pdf/H3_AP202603271820798797_1.pdf) |
| **竞争格局** | 全球市场美国DWA、Alvant等前两大厂商占约70%全球份额；中国市场竞争更分散，主要有有研复材、安徽相邦、中科复材、银邦股份等 [[来源：格隆汇研报]](https://m.gelonghui.com/p/2026609) |
| **主要竞品** | ①安徽相邦（铝基复合材料，未上市）；②中科复材（手机电池仓竞品）；③银邦股份（铝基复合材料）；④DWA等进口替代对象 [[来源：问可汇研报]](https://www.wenkh.com.cn/reports/aluminum-based-composite-material-1914647866552475648) |

| **供需状态** | 军品供给受限（资质壁垒），供需相对平衡；民品（智能终端）受下游折叠机渗透率影响，短期阶段性过剩 |
| **价格趋势** | 军品价格相对稳定（协议价）；民品毛利率已从2022年的26.93%下降至2025H1的24.93% [[来源：招股说明书]](http://money.finance.sina.com.cn/corp/view/vCB_AllBulletinDetail.php?id=11801637&stockid=688811) |

> **行业结论**：`行业整体顺风（国产替代+高端制造升级），但民品竞争加剧，利润池扩张速度不及市场预期；军品属于政策性刚需，增长节奏受制于军工采购计划。`

### 2.3 定价权

| 维度 | 判断 |
| --- | --- |
| **提价能力** | 军品较强（作为唯一供方有一定议价权）；民品弱（大客户强势压价，毛利率下降趋势明显）|
| **顺价能力** | 一般（原材料铝、铜等金属价格上涨时，部分可通过协议转嫁，但民品客户较为强势）|
| **客户议价** | 军工客户相对公平；头部手机厂商议价权极强（苹果/华为等大厂对供应商有绝对议价权）|
| **竞争压力** | 民品端中科复材等竞争者加入，价格战风险存在 |

> **定价权结论**：`中（军品较强、民品偏弱，整体中等）`

### 8.10 一段话总结

> 有研复材是国内金属基复合材料国产替代的稀缺标的，深耕30年、军品独家供方地位壁垒真实，智能终端新业务已成功切入头部手机厂商，未来随IPO二期产能释放和军民需求共振，中长期成长逻辑清晰。但当前公司处于业绩调整期（2025年营收下滑、扣非利润停滞、现金流承压），刚上市叠加新股炒作将估值推至PE 150-200倍的极端高位，远超盈利增速所能支撑的水平；最大预期差在于市场高估了收入增长对利润的传导效率，实际上毛利率在持续下降、期间费用在上升。建议在当前价位（~24元）保持观望，等待估值消化至合理区间（PE 50-80倍，约8-12元）或等到2026下半年军工+折叠屏需求确认性回升时再行介入；若出现重大军品定点公告或折叠屏出货量数据超预期，可作为短期事件驱动交易机会，但不宜作为中长期配置底仓。
