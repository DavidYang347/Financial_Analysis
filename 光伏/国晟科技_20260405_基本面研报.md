## 一、商业模式与竞争优势

### 1.1 业务概述

| 字段 | 内容 |
|------|------|
| **主营业务** | ①**光伏板块**：大尺寸高效异质结（HJT）光伏电池研发、生产、销售；HJT/TOPCon/PERC等组件生产与销售；光伏电站EPC总承包（探索阶段）。②**园林生态板块**：园林工程施工、园林景观设计，形成苗木种植-设计-施工一体化。[[来源：证券之星]](https://stock.stockstar.com/RB2026040100022259.shtml) |
| **收入来源构成** | 2024年光伏业务收入占比约95%以上（营收同比+111.94%主要由光伏组件销售订单增加驱动）；园林业务占比已降至5%以内。[[来源：2024年年度报告]](http://static.cninfo.com.cn/finalpage/2025-04-26/1223331116.PDF) |
| **地理收入拆分** | 以国内销售为主；已在德国、西班牙等分布式场景实现产品落地，通过境内主体出口，境外资产仅867,849元（占总资产0.03%）。[[来源：互动平台/半年报]](https://www.stcn.com/quotes/index/sh603778.html) |
| **核心客户群体** | B端：央企大客户为主（中广核、中国电建等），2024年仅与中广核集团有关订单合计约19.35亿元；少量分布式客户。[[来源：36氪]](https://36kr.com/p/3640012753957639) |
| **收入模式** | 产品销售（电池片+组件）+ 项目制EPC（探索阶段）。光伏为一次性销售为主，园林为项目制。 |
| **价值链位置** | 光伏产业链中游（从切片→电池→组件，垂直布局）；在江苏、安徽、河北、山东、内蒙、新疆六省布局。话语权偏弱：产能规模远小于行业龙头，定价权缺失。[[来源：2024年年度报告]](http://static.cninfo.com.cn/finalpage/2025-04-26/1223331116.PDF) |
| **季节性特征** | 光伏业务具有一定季节性：Q4是组件交付旺季（2024年Q4营收10.28亿元，占全年49%）；园林业务Q4亦为施工旺季。Q1为明显淡季。 |
| **产品/服务生命周期** | HJT电池处于成长期，目前是光伏技术路线角逐阶段；TOPCon已进入快速扩张期竞争白热化；PERC进入成熟期/衰退期。公司HJT布局尚未形成规模优势。 |

### 1.3 行业格局（波特五力分析）

| 字段 | 内容 |
|------|------|
| **行业生命周期** | 光伏行业整体处于成长后期，但产能严重过剩，行业进入深度整合洗牌期（2024-2025年为最痛苦周期底部） |

| 竞争力量 | 强度 | 关键判断依据 |
|---------|------|------------|
| 现有竞争者的竞争强度 | **极高** | 光伏全行业2024-2025年陷入史上最惨烈价格战，组件价格跌破0.7元/W，全行业亏损 |
| 新进入者威胁 | 中 | 重资产壁垒较高，但政策支持已导致前期大量资本涌入，现阶段威胁降低 |

> **行业吸引力综合评估**：光伏行业当前处于极不吸引的阶段——产能过剩、价格内卷、全行业亏损。国晟科技作为小型后进入者，处于最不利的竞争位置，在这一周期中生存压力极大。
