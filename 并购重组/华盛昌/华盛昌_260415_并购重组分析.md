## 一、近期股价上涨原因分析

### 1.1 事件催化：重大资产收购公告

2026年2月28日，华盛昌披露拟以**现金方式收购深圳市伽蓝特科技有限公司（下称"伽蓝特"）100%股权**，整体估值暂定为**4.6亿元**。这是本轮股价异动的最核心驱动因素。

公司股价在2026年4月3日和4月7日连续两个交易日收盘价涨幅偏离值累计超过20%，公司就此发布股票交易异常波动公告，后续4月13日、4月14日相继再度涨停。市场将本次收购解读为华盛昌从传统测量仪器向**AI算力核心配套赛道（CPO光通信测试）**的战略转型。

### 1.2 募投变更提振资金面信心

除并购消息外，公司同步公告**终止部分募投项目**，并将**1.1074亿元募集资金永久补充流动资金**。此举：

- 增强了短期资金流动性，降低资金压力；
- 支付安排采用分期付款方式，减少一次性资金消耗；
- 市场解读为公司在收购资金安排上的主动优化，实施确定性提升。

### 1.3 板块联动与资金共振

- 仪器仪表板块近期受市场关注度持续提升；
- CPO/光通信概念2026年持续维持高热度，相关板块个股轮动活跃；
- 龙虎榜数据显示，4月7日总买入2.73亿元，总卖出2.52亿元，主力资金呈净流入态势；
- 近5日走势持续强于大盘及行业平均。

### 1.4 "抢跑"迹象与市场预期差

值得注意的是，华盛昌在正式发布收购公告前，股价已出现提前涨停的异常波动，市场存在一定的信息提前消化。后续公告落地后，行情得到进一步确认性推动。

## 二、华盛昌主营业务基本面

### 2.1 公司概况

华盛昌成立于1991年，是国内**测量仪器仪表领域的龙头企业**，国家级高新技术企业。主要产品线涵盖：

- 红外热像仪、非接触式人体测温仪
- 颗粒物/空气质量检测仪（PM2.5）
- 激光测距仪、数字万用表、钳形表
- 气体检测仪、甲醛测试仪、噪音计、风速计
- 工业内窥镜、涂镀层测厚仪等

产品广泛应用于电力、暖通制冷、消防、石油化工、轨道交通、环境治理等专业领域，在国内外拥有数百项有效专利。

### 2.2 近期财务表现（承压中修复）

| 指标 | 2025年前三季度 | 同比变动 |
| --- | --- | --- |
| 营业收入 | 5.30亿元 | -5.49% |
| 归母净利润 | 6,489万元 | -40.94% |
| 扣非净利润 | 6,201万元 | -39.92% |
| 经营活动现金流 | 7,643万元 | -48.80% |
| Q3单季营收 | 1.81亿元 | -20.70% |
| Q3单季净利润 | 2,142万元 | -53.11% |

**业绩承压原因：**
- 中美关税升级对出口业务形成冲击
- 研发投入加大，股份支付费用（前三季度2,181万元）拖累账面利润
- 剔除股份支付后，实际净利润约8,670万元，基本面好于表观数据

**修复信号：**
- 越南生产基地规模化投产，2025年9月单月（剔除股份支付）净利润同比企稳，呈逐月修复趋势
- 毛利率维持在43.84%，仍属高毛利率水平
- 资产负债率仅26.09%，财务结构健康，具备出手收购的现金实力

## 三、并购标的：伽蓝特科技深度解析

### 3.1 伽蓝特公司概况

深圳市伽蓝特科技有限公司成立于2008年，是专注于**光通信测试设备**的高新技术企业，总部位于深圳，在北京、武汉、苏州设有销售分支机构。

**核心产品线：**

| 类别 | 主要产品 |
| --- | --- |
| 有源光器件测试系统 | 误码测试仪、高速光电示波器、激光光源、光放大器 |
| 无源光器件测试系统 | 光开关、光功率计、光衰减仪、光谱分析仪 |
| 高端测试平台 | 晶圆级硅光芯片测试系统、CPO测试平台 |
| 速率覆盖范围 | 100G / 400G / 800G 乃至 1.6T 光模块测试 |
| 定制化方案 | 光电控制、光电检测、系统集成 |

**核心竞争力：**
- 覆盖CPO量产环节全流程测试，是高速光模块研发与量产测试的**核心设备供应商**
- 产品技术覆盖从晶圆级芯片测试到系统级模组测试的全链条
- 已进入国内头部光模块厂商供应链，具备替代是德科技、泰克等国际巨头的基础

### 3.2 订单与业绩数据

| 业绩承诺（3年合计） | 净利润不低于1.15亿元（年均约3,833万元） |

### 3.3 交易结构亮点

**亮点1：复合交易结构，强绑定创始团队**

采用"**现金收购 + 实控人协议转让股份**"组合模式：
- 华盛昌以4.6亿元现金收购伽蓝特100%股权，满足股东即时变现需求
- 华盛昌实控人袁剑敏以23.71元/股（前一交易日收盘价90%折扣）向伽蓝特创始人余新文协议转让5%股份（约947万股）
- 余新文所获股份分4批解锁，与3年业绩承诺完成度挂钩，有效防止创始人"套现离场"

**亮点2：股份锁定与业绩对赌深度挂钩**

| 期次 | 解锁条件 | 解锁比例 |
| --- | --- | --- |
| 第一期 | 2026年完成累计承诺业绩33% | 解锁2%股份 |
| 第二期 | 2026-2027年完成累计承诺66% | 累计解锁3%股份 |
| 第三期 | 2026-2028年完成累计承诺100% | 累计解锁4%股份 |
| 第四期 | 股份过户满48个月+业绩补偿义务履行完毕 | 剩余1%股份 |

**亮点3：核心团队激励完善**
- 核心人员签署5年劳动合同 + 竞业协议
- 设置**20%超额业绩奖励机制**：基于三年实际净利润超出承诺部分的20%一次性计提
- 机制设计有效激励团队，避免关键人才流失

**亮点4：无需定增，不摊薄股本**
- 全现金收购，不涉及发行股份
- 财务安排采用分期付款，降低一次性资金压力
- 注册制下审核流程更简洁，实施确定性较高

## 四、战略逻辑：为何此时跨界光通信测试？

### 4.1 行业背景：CPO大年，测试设备需求爆发

2026年被市场定义为**CPO（共封装光学）商用元年**：

- AI大模型训练与推理需求激增，推动数据中心光模块从400G向800G/1.6T快速升级
- CPO因低功耗、高密度优势成为下一代数据中心核心方案
- 光模块速率升级对测试设备的精度、速度、集成度提出指数级要求
- **存量测试设备面临强制更新换代**，新增测试设备需求井喷

### 4.2 国产替代空间：高端市场长期被外资垄断

| 竞争格局 | 详情 |
| --- | --- |
| 现有市场格局 | 高端光通信测试市场长期被是德科技（Keysight）、泰克（Tektronix）等国际巨头垄断 |
| 国产替代进度 | 伽蓝特已凭借技术突破+高性价比进入国内头部光模块厂商供应链 |
| 政策支持 | 科技自立自强背景下，国产测试设备替代进口具备政策红利 |
| 市场空间 | 国内光通信测试设备市场规模持续扩张，国产替代空间广阔 |

### 4.3 产业协同：精准延伸而非盲目跨界

华盛昌并非跨行业并购，而是**产业链内的精准延伸**：

- 华盛昌深耕测量仪器20余年，具备光电测量技术积累
- 伽蓝特专注光通信测试，与华盛昌原有技术形成**高度协同**
- 双方在研发、渠道、客户层面具备整合空间
- "卖铲人"商业模式：不直接参与光模块竞争，而是服务于整个产业链的测试需求，具备**高壁垒、高稳定性**的商业逻辑

华盛昌本次收购伽蓝特是一场**以小博大的战略布局**：以4.6亿元收购一家潜在年化净利润可达2.5亿元的光通信测试资产，同时完成从传统测量仪器向**AI算力核心配套（CPO测试赛道）**的战略转型。

交易结构设计精良：无股权摊薄、强绑定创始团队、超额激励机制完善，并购实施的确定性高于一般重组案例。
