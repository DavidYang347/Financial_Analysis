## 一、商业模式与竞争优势

### 1.1 业务概述

| 字段 | 内容 |
|------|------|
| **主营业务** | 公司专注于**炭黑（碳基纳米材料）** 及**白炭黑（硅基纳米材料）**的研发、生产和全球销售，是国内炭黑行业前三位企业。炭黑粒子尺寸在11～100nm，主要用于橡胶补强，广泛应用于轮胎、橡胶制品、新能源、油墨涂料等领域。[[来源：2024年年度报告摘要]](http://notice.10jqka.com.cn/api/pdf/ed88b0f8ba3aa150_1743510430/%E9%BE%99%E6%98%9F%E7%A7%91%E6%8A%80%EF%BC%9A2024%E5%B9%B4%E5%B9%B4%E5%BA%A6%E6%8A%A5%E5%91%8A%E6%91%98%E8%A6%81.pdf) |
| **收入来源构成** | 炭黑为绝对主体业务（占营收约95%+），白炭黑及煤焦油制品为辅助。2024年炭黑板块毛利率8.72%，同比略升0.74pct。[[来源：国海证券研报]](https://m.hibor.com.cn/wap_detail.aspx?id=610ab8d4eff7955614b7fd1eb7e02a90) |
| **地理收入拆分** | 国内：37.49亿元（占比86.07%，同比+2.83%）；海外：6.07亿元（占比13.93%，同比-3.2%）。出口市场主要集中在欧洲、东南亚，客户涵盖米其林、固特异、普利司通、大陆马牌、韩泰等国际轮胎巨头。[[来源：中国能源网报道]](https://www.cnenergynews.cn/ssgs/2025/04/03/detail_20250403207661.html) |
| **核心客户群体** | B端为主：下游为轮胎企业（风神轮胎、中策集团、玲珑轮胎、赛轮轮胎等国内龙头，以及米其林、大陆马牌等外资），以及油墨、涂料、塑料等非橡胶行业用户。[[来源：同花顺F10]](http://basic.10jqka.com.cn/002442/) |
| **收入模式** | 一次性销售为主（炭黑产品直销/批量采购），部分大客户签订长期框架协议。非订阅制，收入与产销量直接挂钩。 |
| **价值链位置** | 产业链**中游**：上游为原材料（煤焦油、蒽油、乙烯焦油等原料油），下游为轮胎/橡胶制品厂商。公司具备一定的规模议价权，但整体话语权偏弱——上游原料价格波动对毛利率冲击显著，下游轮胎大客户采购集中度较高。 |
| **季节性特征** | 具有一定季节性：下游轮胎行业受国内公路运输淡旺季影响，Q4受全钢胎需求回落和年末库存消化影响较为明显。2024年Q4营收12.32亿元为全年最高（受益于山西新产能爬坡），但净利润环比Q3有所回落。[[来源：国海证券研报]](https://m.hibor.com.cn/wap_detail.aspx?id=610ab8d4eff7955614b7fd1eb7e02a90) |
| **产品/服务生命周期** | 传统橡胶炭黑处于**成熟期**，需求相对稳定但价格竞争激烈。碳基新材料（高纯度纳米炭黑用于新能源、电缆等）处于**成长期**，是公司未来重要增量方向。 |

> **护城河宽度评级**：**一般**（接近窄护城河）
>
> **分析说明**：龙星科技的核心竞争力在于**规模 + 成本 + 品牌**三位一体，在国内炭黑细分行业具备较强竞争力，产能利用率常年超100%是其经营效率较高的体现。然而炭黑本质上是高度同质化的化工品，护城河深度有限，定价权受制于上游原料价格波动及下游大客户议价。向碳基新材料（高纯纳米炭黑）转型若能成功，护城河有望加深；目前该转型仍处早期，尚未形成实质性竞争壁垒。

### 1.3 行业格局（波特五力分析）

| 字段 | 内容 |
|------|------|
| **行业生命周期** | 传统橡胶炭黑：**成熟期**；碳基新材料炭黑（特种/高纯纳米炭黑）：**成长期** |
| **市场集中度** | 行业集中度中等偏低。2024年全国会员企业炭黑生产总量512万吨。前三强（黑猫股份约94万吨、金能科技约65万吨、龙星科技约55.5万吨）合计约占会员总产量约42%。[[来源：中国能源网]](https://www.cnenergynews.cn/ssgs/2025/04/03/detail_20250403207661.html) |
| **主要竞争对手** | ①**黑猫股份**（行业第一，产销约94万吨，规模优势最强）；②**金能科技**（行业第二，产销约65万吨）；③**永东股份**（特种炭黑为主，毛利率相对较高）。 [[来源：国海证券研报]](https://m.hibor.com.cn/wap_detail.aspx?id=610ab8d4eff7955614b7fd1eb7e02a90) |
| **潜在替代威胁** | 在轮胎补强领域，白炭黑（沉淀法二氧化硅）存在一定替代趋势（绿色轮胎需求），但替代进程缓慢，炭黑短期地位稳固。新能源电池、线缆等非轮胎领域使用量持续提升，为炭黑打开新需求空间。 |
| **公司市场份额** | 约占全国会员企业炭黑总产量9.94%，产销量排名国内重点企业前三位，近2年趋势**上升**（受益于山西产能投放）。[[来源：同花顺F10]](http://basic.10jqka.com.cn/002442/) |

**波特五力量化评估**：

| 竞争力量 | 强度 | 关键判断依据 |
|---------|------|------------|
| 现有竞争者的竞争强度 | **高** | 国内炭黑企业众多，产品高度同质化，价格竞争激烈；2025年因原料成本波动叠加下游需求走弱，行业整体盈利大幅收窄 |
| 新进入者威胁 | **中** | 炭黑生产需较大固定资产投入（窑炉设备）及环保资质，有一定门槛，但门槛并非不可逾越 |
| 替代品威胁 | **中** | 白炭黑在绿色轮胎中存在部分替代，但炭黑短期仍不可替代 |
| 供应商议价能力 | **高** | 核心原料煤焦油价格受国际油价、焦化行业景气周期影响大，波动剧烈，是利润率最大不确定性来源 |
| 买方议价能力 | **中高** | 下游米其林、大陆马牌等国际轮胎巨头采购规模大，议价能力强；国内轮胎厂集中度较高，有一定压价能力 |

> **行业吸引力综合评估**：炭黑是典型的**重资产、强周期、低利润率**行业，行业结构不算吸引人。龙星凭借规模、成本和客户资源处于行业竞争前列，但整体行业天花板清晰、盈利能力受原材料-产品价格双重波动挤压。向碳基新材料的转型是未来核心看点，但当前贡献微薄。

### 11.3 消息面催化因子

**正面催化（做多信号）**：

- 中东地缘局势推动油价上涨，炭黑行业价格中枢存在向上修复预期
- 国家电网5万亿投资规划的逐步落地，高端线缆炭黑需求持续增长
- 中橡（重庆）产能并入完成，西南市场开拓有望贡献新增量

**负面压制（做空信号）**：

- 2025年全年业绩预计同比下降50%以上，业绩大幅恶化拖累估值
- 实控人减持可转债，传递出一定负面预期信号
- 行业供给仍在扩张，成本传导机制短期未见修复

**待验证的关键事件**：

- 2026年一季报业绩：能否出现利润同比明显回升？
- 炭黑价格能否在2026年受油价上涨驱动持续走强？
- 碳基新材料（纳米炭黑）的商业化收入能否在2026年取得突破？

### 11.4 当前核心看点总结

**看点一：产能扩张完成后的规模效益释放**
> 2025年山西一期20万吨产能已完成竣工验收，中橡重庆8.5万吨并入，公司炭黑产能将超过70万吨、有望突破90万吨，成为行业第二。随着新产能爬坡完成，在行业景气改善时，规模效益将集中释放，盈利弹性较大。但**当前产能释放恰逢行业价格低谷**，时间窗口错配，需等待行业供需出现积极信号。

**看点二：炭黑行业价格底部反转预期**
> 2025年受多重压力炭黑行业盈利骤降，但中东地缘局势带动油价大幅上行，煤头路线的成本优势凸显，炭黑价格存在触底反弹的可能性。若价格中枢上移1000元/吨，以70万吨产量测算，可带来约7亿收入增量、0.3-0.5亿净利润弹性，对股价催化明显。**这是当前最重要的短期博弈逻辑，但确定性有待观察。**

**看点三：碳基新材料长期成长期权**
> 公司更名"科技"、布局高纯纳米炭黑、建设碳基新材料研发中心，与复旦、中科院等开展产学研合作，叠加国家电网5万亿投资对高端炭黑的长期驱动，具备一定的成长叙事。但**目前新材料尚无实质收入贡献**，属于远期期权，市场对此尚未充分定价。
