## 一、商业模式与竞争优势

### 1.1 业务概述

| 字段 | 内容 |
|------|------|
| **主营业务** | 三大业务板块：①锂电新能源原料开发与利用（锂矿开采、锂盐生产销售）；②稀有轻金属（铯、铷）资源开发与利用；③固体矿产勘查和矿权开发。2024年新增铜矿采选冶业务（赞比亚Kitumba铜矿+纳米比亚Tsumeb冶炼厂）。 [[来源：2024年报摘要]](https://file.finance.qq.com/finance/hs/pdf/2025/04/25/1223271313.PDF) |
| **收入来源构成** | 2024H1锂业务毛利占总毛利约56%，铯铷业务约34%。2024全年铯铷板块收入13.95亿元（+24.16%），锂盐板块收入约39亿元（估算，全年总收入53.64亿元），固体矿产勘查业务收入占比持续收缩至个位数。[[来源：国信证券研报]](https://pdf.dfcfw.com/pdf/H3_AP202411191640961288_1.pdf) |
| **地理收入拆分** | 高度国际化：矿山资产分布于津巴布韦（Bikita）、加拿大（Tanco）、赞比亚（Kitumba）、纳米比亚（Tsumeb），锂盐生产和铯铷精细化工在中国江西，甲酸铯运营中心在英国/挪威，业务遍及欧洲、亚太、中东、北美。境外收入占比估算超60%。[[来源：2024年报摘要]](https://file.finance.qq.com/finance/hs/pdf/2025/04/25/1223271313.PDF) |
| **核心客户群体** | B端工业客户为主：锂盐下游为动力电池/正极材料企业（已进入特斯拉供应链）；铯铷盐下游覆盖油气公司（甲酸铯钻井液）、半导体/特种玻璃/科研机构（铯铷盐精细化工）。[[来源：2024年报摘要]](https://file.finance.qq.com/finance/hs/pdf/2025/04/25/1223271313.PDF) |
| **收入模式** | 产品销售为主（锂盐、铯铷盐按吨销售）；甲酸铯采用"生产+租售+技术服务+回收提纯"生态模式（具备一定订阅/服务收入属性）；勘查技术服务为项目制。[[来源：2024年报摘要]](https://file.finance.qq.com/finance/hs/pdf/2025/04/25/1223271313.PDF) |
| **价值链位置** | 锂业务：采选冶一体化，覆盖上游矿山到锂盐成品，属产业链上中游，拥有较强话语权；铯铷业务：从采矿到精细化工终端客服，全产业链龙头，垄断性地位；铜业务：尚在建设期，规划采选冶一体化。[[来源：2024年报摘要]](https://file.finance.qq.com/finance/hs/pdf/2025/04/25/1223271313.PDF) |
| **季节性特征** | 矿业整体季节性较弱；甲酸铯与油气钻井活动相关，北半球冬季略有波动；锂盐受下游新能源车消费节奏影响，Q4相对旺季。整体季节性不显著。 |
| **产品/服务生命周期** | 锂盐：新能源成长期，但供给大幅扩张导致价格周期性下行；铯铷盐：成熟稳健，高端应用持续拓展；铜：初入阶段；锗/镓：战略性稀缺资源，导入期。 |

### 1.2 核心护城河评估

| 护城河类型 | 是否具备 | 评估说明 |
|-----------|---------|---------|
| 品牌壁垒 | ✅ | 铯铷领域全球知名，甲酸铯市场绝对领先，锂盐进入特斯拉供应链 |
| 成本优势 | ✅ | 自有矿山比例大幅提升（2024年锂资源自给率超70%），光伏自发电降成本，开采成本具竞争力 |
| 网络效应 | ❌ | 矿产资源行业网络效应不适用 |
| 转换成本 | ✅（铯铷强/锂弱） | 甲酸铯技术服务生态高度绑定客户，回收再利用体系形成闭环；锂盐下游转换成本弱 |
| 专利/资质壁垒 | ✅ | 透锂长石单烧及混烧工艺、铷盐萃取工艺已获发明专利；铯资源进入门槛极高（全球稀缺） [[来源：2024年报摘要]](https://file.finance.qq.com/finance/hs/pdf/2025/04/25/1223271313.PDF) |
| 规模经济 | ✅ | Bikita矿山418万吨/年选矿产能，锂盐6.6万吨/年产能形成规模优势 |
| 数据壁垒 | ✅ | 二十余年海外地勘经验积累，找矿数据和技术积累难以复制 |
| 生态系统绑定 | ✅（铯铷业务） | 甲酸铯"生产-租售-技术服务-回收"闭环，客户深度绑定 |
| 地理/渠道垄断 | ✅ | 铯铷：Tanco和Bikita矿山是全球最主要铯资源，接近垄断供给 |

> **护城河宽度评级**：铯铷业务**宽护城河**；锂业务**窄护城河**（竞争激烈）；铜矿业务尚待验证
>
> **分析说明**：公司最核心、最持久的护城河在于铯铷全球垄断资源与产业链。锂业务虽有自给率优势，但处于供过于求周期，价格持续下行压制盈利，护城河宽度有限。铯铷业务毛利率高达72%，具备真正意义上的定价权，且客户粘性强、壁垒高，是公司价值的核心锚点。

### 1.3 行业格局（波特五力分析）

| 字段 | 内容 |
|------|------|
| **行业生命周期** | 锂：成长期（新能源驱动，但供给过剩导致价格周期底部）；铯铷：成熟期（高端稀缺品，稳健增长）；铜：成长期（绿色转型需求旺盛） |
| **市场集中度** | 铯铷：公司占全球铯产量绝对主导地位（接近垄断）；锂精矿：全球主要供应来自澳大利亚Greenbushes、PILBARA、SQM等，公司Bikita矿山已跻身重要供应商 |
| **主要竞争对手** | 锂：赣锋锂业（002460）、天齐锂业（002466）、雅化集团（002497）；铯铷：无实质性全球竞争对手（Cabot特殊流体原为主要对手，已被收购） [[来源：国信证券研报]](https://pdf.dfcfw.com/pdf/H3_AP202411191640961288_1.pdf) |
| **潜在替代威胁** | 锂：钠离子电池长期存在替代风险；铯铷：无近期替代威胁，用途高度专业化；铜：电气化需求是结构性驱动力 |
| **公司市场份额** | 铯铷：全球甲酸铯绝对市场主导；锂精矿：约占全球供应2-3%，属中小份额参与者，近年趋势上升 |

**波特五力量化评估**：

| 竞争力量 | 强度 | 关键判断依据 |
|---------|------|------------|
| 现有竞争者的竞争强度 | 锂：高 / 铯铷：低 | 锂盐产能集中释放引发价格战；铯铷无有效竞争者 |
| 新进入者威胁 | 低 | 矿山开发周期长、资本投入大；铯铷资源极度稀缺，无法复制 |
| 替代品威胁 | 锂：中 / 铯铷：低 | 锂电面临钠电、固态电池长期演变；铯铷化学特性不可替代 |
| 供应商议价能力 | 低（自给） | 公司自有矿山高度整合，外购矿依赖持续下降 |
| 买方议价能力 | 锂：高 / 铯铷：低 | 锂盐买方选择多、议价能力强；铯铷买方依赖单一供应商 |

> **行业吸引力综合评估**：铯铷业务所处细分行业高度垄断、壁垒极高、盈利极佳，是公司价值的压舱石。锂业务行业吸引力当前处于低谷，供给过剩格局短期难以扭转，盈利弹性取决于锂价复苏节奏。铜矿业务所在行业长期供需偏紧，战略价值大，但短期投产尚需时日。整体来看，公司布局的品种具备较好的长期战略价值，当前处于锂价周期底部的盈利低谷期。

### 1.4 定价权分析

| 维度 | 内容 |
|------|------|
| **历史提价记录** | 铯铷业务：在全球需求扩张背景下，铯铷盐价格持续上涨，2024年铯铷板块收入同比增长24%、毛利同比增长51%，体现显著提价能力。锂盐：跟随市场价格，无主动提价能力，2022-2024年均价从60万元/吨跌至约10万元/吨区间。[[来源：2024年报摘要]](https://file.finance.qq.com/finance/hs/pdf/2025/04/25/1223271313.PDF) |
| **成本转嫁能力** | 铯铷：强，可将矿石成本上涨充分转嫁，甲酸铯毛利率长期维持70%以上；锂：弱，原料+电力成本压力只能靠内部降本而非提价消化 |
| **产品溢价水平** | 铯铷：相对全球市场存在一定定价权，甲酸铯按租用服务定价；锂盐：高纯、低杂质产品相较普通品略有溢价，但整体随市定价 |
| **提价阻力** | 锂：极强，来自买方议价和竞争对手压价；铯铷：弱，客户依赖公司供应 |
| **定价权综合评级** | 铯铷业务：**强**；锂业务：**弱** |

> 这一隐含假设在**乐观情景**下是合理的（铜/锗按时放量+锂价2026年显著回升），但在**中性/悲观情景**下则**过于乐观**。

### 8.4 估值结论

> 中矿资源当前估值较为激进。TTM PE高达147x，Forward PE在35-70x区间（视2025-2026年业绩恢复程度而定），PB约5x。铯铷垄断业务给予溢价合理，但当前股价更多是对铜/锗/多金属布局期权价值的定价。卖方机构目标价均值55.67元已低于当前股价85元，暗示在中性假设下可能存在较大下行风险。核心投资逻辑依赖于：①锂价周期触底回升；②Kitumba铜矿和Tsumeb锗项目按时投产。若以上两条均兑现，当前估值或具备支撑；若任一延迟，股价回调风险较大。

### 12.3 核心投资逻辑

> 中矿资源的核心投资逻辑在于：**以铯铷全球垄断业务为压舱石，以锂价周期底部反转为弹性来源，以铜/锗/镓多金属布局为第三曲线期权**，是一家资源储量丰富、技术壁垒深厚的多金属矿业平台公司。当前股价85元已定价较为乐观的情景（2026年业绩大幅修复+铜/锗投产），具备一定博弈属性，适合能承受较高估值风险的投资者。

**逻辑成立的关键前提**：
1. 铯铷业务保持年均10%+增速，毛利率维持70%以上
2. Kitumba铜矿和Tsumeb锗项目按计划推进，2026-2027年形成实质性利润贡献
3. 碳酸锂价格在2025-2026年触底并回升至10万元/吨以上水平

### 12.4 主要风险提示

| 风险 | 触发信号 | 若触发的应对动作 |
|------|---------|---------------|
| 锂价持续下探至5万元/吨以下 | 碳酸锂期货主力合约跌破5万元/吨且持续超过2个月 | 重新评估，降低锂业务盈利预期，下调目标价 |
| Kitumba铜矿建设延期超12个月 | 未找到管理层公告原计划建设节点，若超期6个月以上则触发 | 重新评估多金属布局期权价值，下调估值上限 |
| 津巴布韦政府实质性封锁Bikita矿山出口 | 政府正式文件禁止Bikita输出，且沟通无进展超过3个月 | 清仓或大幅减仓，该风险触发将对公司基本面造成重大冲击 |
